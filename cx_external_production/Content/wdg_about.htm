Digital River Widgets Overview

Digital River is dedicated to providing the tools and features you need to enhance your store and improve the shopping experience for your customers. Our dedicated Site Design team has created a collection of JavaScript widgets you can use to add product and other content to your store. These widgets are useful for building dynamic, user-friendly stores for your shoppers—helping you create a store that stands above your competition.

What are widgets?

A widget is a script URL that you add to a page on your store to insert special functionality or objects on the page.

Note: Implementing widgets requires some knowledge of JavaScript, XML, CSS, and HTML. Those who have technical knowledge should be able to implement these widgets and customize them using the optional parameters and variables. If you need assistance, contact your Store Operations team.

There are several types of widgets available from Digital River that allow you to do various things with the content and data on your store:

  • Mini Cart Widget v2 – Shows a mini version of the shopping cart on the page where the URL for the widget is positioned.

Supported browsers

Most of the DR widgets use standard JavaScript and are compatible with any standard HTML page. Digital River has tested each widget in Microsoft© Internet Explorer© 7.0 and Firefox© and the widgets display as expected in these browsers. The Mini Cart widget v2 does not work with Safari© 2 because it uses client-side XSLT but might work in Safari 3.

Output options

Certain widgets allow you to set the output to JSON or XML, depending on your requirements. To set the output, define the "output" parameter in the sample scripts or URLs. To request JSON data across domains, use the JSONP parameter to wrap the response in a callback function.

Virtual domains

If your store uses a virtual domain, you must change "store.digitalriver.com" in the widget scripts to your virtual domain.
