Configuring the Import and Export Product Spreadsheet

You can set up which product attributes should appear in the product Import and Export spreadsheet template. Configuring which fields appear in the product spreadsheet template can make setting up products in the spreadsheet easier and faster, and reduce the amount of time it takes to process the spreadsheet during an import or export.

Note: It is strongly recommended that any configuring done to the Import and Export template also be done to the Product Setup (see Configuring the Product Template for more information). This will make the spreadsheet fields match what fields are seen in the UI. This will reduce confusion and possible configuration errors with products that are created or editing using both methods.

To configure the import and export spreadsheet:

  1. On the Catalog menu, select Products, and click Import and Export Products. The Products Import and Export page appears.
  2. Click Configure Product Import & Export in the taskbar. The Configure Product Import and Export Spreadsheet page appears.
  3. Three tabs appear on the page. Click the tab for the attributes you want to configure. Then, select or clear the check boxes for the attributes you want to include (or exclude) from the spreadsheet template.
    • To include an attribute, select the check box next to the attribute name. The attribute will appear in the Import & Export spreadsheet.
    • To remove an attribute, clear the check box next to the attribute name. The attribute will not appear in the Import & Export spreadsheet.
    • Attributes that are grayed out and marked with an asterisk (*) are required attributes and cannot be removed from the spreadsheet.
    • Click the Select/Clear All check box to select all or clear all attributes in a particular section.
  4. When you are done configuring the spreadsheet, click Save.
