Exporting Offers

You can export offer data using the Marketing Import and Export tool. When you export offers you can enter the offer IDs or product IDs for the offers you want, or specify the date range for offers. If needed, you can filter the exported offers by status.

The blank template you can export is a blank version of the spreadsheet you get when you export offers. It contains all the fields and attributes that you set for an offer and is a good reference when creating or working with offers. You can use this file to create new offers by entering the data in the spreadsheet and then importing that file to the system.

To export offers:

Role Needed: Merchandising ManagerA user role that provides access to various merchandising features, including offers and tracking programs. Search this file for the "user role definitions" topic for more information. and Merchandising Discount ManagerA user role that allows the user to set the discount amount or discount percentage in marketing offers. Merchandising Discount Managers cannot create new offers, just set the discount for an existing offer. To create an offer you would also need the Merchandising Manager role.
  1. On the Marketing menu, select Offers and Programs, and click Import and Export Offers. The Offers Import and Export page appears.
  1. Select an Export Option. This determines what you want to export or how you want to define the offers you want to export.
    • Select Blank Template if you want to export a blank offer template that you can use to create new offers.
    • Select Specify Offer IDs if you want to enter the IDs for the offers you want to export. You can enter more than one ID.
    • Select Specify Product IDs if you want to export offers for a specific product. You can enter more than one ID.
    • Select Specify Date Range if you want to export offers that will end or have ended with a time frame you specify. When you select this option, the Start Date Range and End Date Range fields appear. Offers that end within this range will be exported. This means that an offer with an end date outside the range you specify will not be exported (the Start Date does not matter). Offers that do not have an end date will also not be exported.
  2. Depending on what you selected in the step above, enter either the Offer ID(s), Product ID(s), and/or Start Date Range and End Date Range for the offers you want to export. Separate offer and product IDs with a comma.
  3. To filter the offers in the export by status, from the Offer Status list, select the status of the offers you want.
    • Open – Offer has been created, but has not been deployed.
    • Deployed – Offer is active on your storefront.
    • Expired – Offer is no longer valid, because the specified end date has been exceeded.
    • Retired – Offer is no longer valid, because it was terminated.
  4. From the File Type list, select whether you want to view offer data for Individual Products or Parent & Child Products. If you know the offers you are exporting contain Parent and Child products you should export that type of file. For example, the Buy M, Get N, Cross-sell, and Up-sell offer types use Parent and Child products in the offer.
  5. Click Export. When prompted, save the file on your computer. You can now make changes to the offer data in the exported file and upload those changes to the system.
