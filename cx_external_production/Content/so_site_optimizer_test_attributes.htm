Site Optimizer Test Attributes

The following table defines the attributes that you will see when you view your list of Site Optimizer tests.

Attribute

Definition

ID

The ID for the test. When you create a test, the ID is assigned to the test by the system.

Name

The name for the test. When you create a test, you enter the name you want the test to have.

Type

The type of test. For information about the types of tests that are available, see Offer Types and Test Types.

Status

The status of the test. Site Optimizer tests can have one of the following statuses:

Pending Start – The test has not yet begun on your store.

In Progress – The test is live and running on your store.
Pending Final Action – The test has ended (you chose to manually end the test) or expired (the End Date was reached). You can choose to promote a level to the store, or terminate the test.
Retired – The test is retired and ended, no data is being collected on the test.
Promoted – The test level is a fully deployed, non-test offer on your site. When one test level is selected for promotion, the other test levels in the test are retired.

Complete Expired – The test has run through its end date. When a test is expired it still appears on the store but reporting for the test has ended.
Completed – The test has been terminated (and no levels have been promoted to the store).

Complete Terminated – The test was ended before it ran through its end date.

Complete Promoted – The test has ended and one of the test cells were promoted to the store.

Complete Retired – The test ended without a winner being promoted.

Start Date

The start date for the test.

End Date

The end date for data collection of the test. Tests with end dates will continue to run beyond the end date until they are promoted, retired, or ended manually. The end date is only used for gathering of data on the test.

Last Modified On

The date when the test was last modified.

What is the difference between "terminated" and "ended?"

When viewing a Site Optimizer Test, you will have the option to "terminate" the test or "end" the test, depending on the test status.

You can end a test when the test status is "In Progress." When an in progress test is ended, the status then changes to "Pending Final Action" and the test will wait for you to promote or terminate the test.

You can terminate a test when the test status is "Pending Final Action" and the test status will change to "Completed" and no other action will be required for the test. If you do promote a level in a Completed test, the status will change to Completed (Promoted).

Site Optimizer Test Cell Attributes

When you view the details for a single Site Optimizer test you will see a summary of the cells within the test. The following table defines the attributes listed for each test cell in a Site Optimizer test.

Attribute

Definition

Name

The name of the test cell.

Offer ID

The ID for the offers associated with the test cell.

Split

The percentage of traffic, also known as the split, that will see the test cell.

Control

Indicates whether the test cell is the control cell in the test.

Unique Visits

The number of unique visits for the test cell.

Sales

The total sales for the test cell.

Orders

The number of orders for the test cell.

Close Ratio

The percentage of traffic assigned to the test cell that completed an order.

Revenue per Visitor

The average revenue generated per unique visitor that sees the test cell.

Confidence

The confidence rating for the test cell. See Confidence Rating for more about how this is calculated.
