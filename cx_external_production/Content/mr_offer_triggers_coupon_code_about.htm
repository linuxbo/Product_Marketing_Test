About Offer Triggers: Coupon Code

Offers that are triggered by Coupon Code require a shopper to enter a special code in the shopping cart to redeem the offer. The coupon codes shoppers must enter can be distributed in a variety of ways such as through email campaigns, postal mailings, or advertisements seen on your store. To learn how to create an offer that uses the coupon code trigger, see Creating Coupon Code Offers.

Offers that are triggered by coupon code can be further restricted by the number of times each code can be used. This is especially useful when you limit each code to a single use and target shoppers with those unique codes, or limit the total code use to a specific number to encourage shoppers to act fast to redeem an offer.

The following offers can be triggered by coupon code:

  • All Products Bundle
  • Banner
  • Bundle
  • Buy M, Get N
  • Discount

Tip: Offers that use the "Coupon Code" trigger must also use the "PointOfSale_CouponEntryField" POP for the coupon code field to appear during checkout.

Getting Coupon Codes

When you create an offer that uses coupon codes, you can have the system create the coupon codes using a set of parameters you define. If you already have a set of codes you want to use, you can import them into the offer.

When an offer uses coupon codes, you can view the number of codes associated with the offer or the number of codes that have already been used. This allows you to easily manage your code inventory and keep the promotion running smoothly on your store.

Tip: We recommend you create, upload, or generate coupon codes for your offer(s) before you distribute any codes to shoppers. This is because system-generated codes are 9-digits and no other codes (uploaded or created) can be 9-digits. If you create your codes for the offer before you distribute any codes to shoppers, you can be sure that the codes will work with the offer.
