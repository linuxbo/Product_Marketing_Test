Importing Offers

You can import new or changed offer data (including images for offers using a Zip® file) using the Marketing Import and Export tool. When you export offers, the offer data is exported in a spreadsheet that you can make changes to. When you are done making changes you can import that data to the system and deploy the offers to have them appear on your site. You can also create new offers by entering offer data directly into a blank template and importing that file.

When you import an offer it is not automatically deployed. Imported offers are given the "design" status and must be deployed before you can use them on your store.

To import offers:

Role Needed: Merchandising ManagerA user role that provides access to various merchandising features, including offers and tracking programs. Search this file for the "user role definitions" topic for more information. and Merchandising Discount ManagerA user role that allows the user to set the discount amount or discount percentage in marketing offers. Merchandising Discount Managers cannot create new offers, just set the discount for an existing offer. To create an offer you would also need the Merchandising Manager role.
  1. On the Marketing menu, select Offers and Programs, and click Import and Export Offers. The Offers Import and Export page appears.
  2. In the Import Offers area, click Browse next to the Offers field. Then, select the spreadsheet file with the offer data you want to import.
  3. To import offer images via a Zip® file, in the Import Offers area, click Browse next to the Offer Images field. Then, select the Zip file with the images you want to import.
  4. Click Import. The page refreshes and a message appears when the import is successful. When offers are imported they are saved in the "design" status meaning they must be deployed before they can used on your store. See Deploying Offers for more information.
