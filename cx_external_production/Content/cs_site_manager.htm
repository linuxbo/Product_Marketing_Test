Viewing Sites with Site Manager

You can search and then view specific sites using the View Site by Locale feature. You can search for sites by name or ID.

To view a site by locale:

  1. On the Customer Service menu, select Tools, and click Site by Locale. The View Site by Locale page appears.
  2. To view a specific locale for the site, in the Locales for Site area, click on the Available Sites link for the locale you want to view. If the locale you want to view is not shown in the list it means that locale is not used by the currently selected site. You can change the site you are viewing by entering the site ID or name and searching for the new site in the fields provided.
  3. To search for another site that you want to view, in the Site Selector area, enter the Site Name or Site ID you want to view and click Search. When the results appears, select the locale for the site you want to view by clicking the locale link.
