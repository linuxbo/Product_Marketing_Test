Editing Shoppers

The Shopper Details page appears when you search for shoppers and click a shopper's Login ID on the Shopper Search Results page. To see all of the previous orders for the shopper displayed on this page, click View All Orders in the taskbar.

To edit shoppers:

  1. On the Customer Service menu, select Search, and click Search Shoppers). The Search Shoppers page appears.
  2. Enter criteria in the Login ID, First Name and Last Name, or E-mail Address fields for the shopper you want to edit. The criteria you enter in these fields must match a shopper exactly, except for the First Name field. In this field, you can enter just the first few letters of a shopper's first name to find matches.
  3. Click Search. The page refreshes and the search results appear on the Shopper Search Results list page. Click on the Login ID link for a shopper to edit. The Shopper Details page appears.
  4. To change login information:
    1. To change the shopper's ID, type a new login ID in the Shopper Login box.
    2. To change the shopper's Password, type a new password in the Password box, and then retype the password in the Confirm Password box. For security purposes, after you click Save Changes, the password you typed is encrypted and appears longer than it actually is.
    3. To deactivate the current login, next to Status, select Inactive. Deactivating a shopper does not delete the shopper from the system.
  5. To change shopper information:
    1. Type a new name in the First Name, or the Last Name box.
    2. Type a new email address in the Email Address box.
    3. Click Update Email Address on All Orders By This User to make any change you make to the shopper's email address effective on every order for this shopper.
  6. To change the user's default address, change the information in the fields provided.
  7. To change additional addresses, change the information in the fields provided. To make one of the shopper's alternate addresses the default address, click Edit, and then click Use as Default Address.
  8. To change the user's credit card information, change the information in the fields provided. You can change the User's Default Credit Card information or the user's Additional Credit Cards information.
  9. When you are done making changes, click Save. Click Reset Values to reset the fields to their last saved values.
