Creating Shipping Offers for Individual Products

You can create a shipping discount offer to give a discount on shipping costs for specific products.

To create or edit a shipping discount for specific products:

Role Needed: Merchandising ManagerA user role that provides access to various merchandising features, including offers and tracking programs. Search this file for the "user role definitions" topic for more information. and Merchandising Discount ManagerA user role that allows the user to set the discount amount or discount percentage in marketing offers. Merchandising Discount Managers cannot create new offers, just set the discount for an existing offer. To create an offer you would also need the Merchandising Manager role.

  1. On the Marketing menu, select Offers and Programs, and click Create Offer. The Create Offer page appears.

  2. In the Name and Description fields, enter a name and, if needed, a description for the offer. Names are required but descriptions are not. Neither are seen by shoppers on your site.

  3. In the Expiration area, select whether you want the offer to begin and/or end on particular dates. When you select Start Date or End Date, fields appear so you can choose the specific date and/or time you want to set. See example.

    In the example below we have set the offer to be available for the first two weeks of September, 2007. Note that we set the time so that the offer begins at midnight of the start date, and ends right before midnight on the end date.

  4. In the Type area, select Discount shipping on individual product.

  5. In the Trigger area, select how you want the offer to be triggered. The trigger determines how the offer is redeemed by shoppers on your site. Click here to see definitions for the trigger options.

    • Always Triggered – Offer available to all shoppers on your site.
    • Promotional URL – Offer is available when shoppers click a URL (in an email, on your site, etc.).
    • Coupon Code – Offer is available when shoppers enter a coupon codeUsed with marketing offers. A code entered on the Shopping Cart page that provides a discount for that shopper. A coupon can be applied to all products or particular products. during checkout.
    • Site Test – Offer will be used in a Site Optimizer1.) A feature in the Command Console that allows users to conduct site-wide A/B testing on a live site. 2.) A user role that provides access to Site Optimizer. Search this file for the "user role definitions" topic for more information. test.

  6. Click Next. The Behavior step appears.

  7. If you chose to trigger the offer by coupon codeUsed with marketing offers. A code entered on the Shopping Cart page that provides a discount for that shopper. A coupon can be applied to all products or particular products., the Coupon Codes area appears so you can choose how coupon codes will be added to the offer.

    From the Type list, select how you want to add coupon codes to the offer. You will add coupon codes to the offer after you create the offer in the Offer Wizard. Click to see explanations of the types. Select the Limit Individual Coupon Usage check box if you want to limit the number of times a single coupon can be used. Then enter the number of times each code can be used in the Coupon Limit field that appears.

    • Enter Coupon Codes – Select this option if you want to enter coupon codes.
    • Generate Coupon Codes – Select this option if you want the system to generate a set of random coupon codes for you.
    • Range – Select this option if you want the system to generate a range of coupon codes for you. The codes generated for this option can have a specific prefix and you can specify the first and/or last number you want to use for the range.
    • Import Coupon Codes – Select this option if you want to import a .txt file that contains coupon codes.

  9. In the Offer Controls area:

    • Select the Limit Total Usage check box if you want to limit the total number of times this offer can be redeemed by shoppers on your site. Then, enter the limit in the Total Limit field that appears.
    • If you chose to trigger the offer by promotional URL or coupon code, the Limit Individual Shopper Usage check box appears. Select this check box if you want to limit the number of times a single shopper can redeem the offer. Then, enter the limit in the Shopper Limit field that appears.
    • For the Locale Restrictions option, select whether you want to restrict the offer on certain locales. If you choose to limit the offer to certain locales, the Locales area will appear further down the page and you will define those locales in a later step of this topic.

  10. More about POPs and the types you can use.

    The points of promotion (POPs) you select for an offer determine where and how the offer appears on your site. POPs have rules about their behavior and what occurs when a shopper encounters the POP.

    For example, cross-sell and up-sell offers are often set up to use an "interstitial" POP that make another window appear when a shopper adds a product to the shopping cart. This interstitial page presents the shopper with additional products they can purchase in addition (or in lieu) of the product they just added to the cart.

    Your site has several standard POPs that are associated with the standard set of offer types. Site design can add new POPs for any store. A POP can also be used to display content without a product offer.

    The following table outlines which offers are available on all sites, with their most commonly associated POPs.

    POP Type Offer Type Description
    First-time visitor pop-up Discount A pop-up for first-time visitors giving them a percentage off their order.
    Shopping Cart Banner A graphic or text banner appears on the Shopping Cart page.
    Point of Sale Coupon Activates the coupon field is provided on the Shopping Cart page.
    Don't Leave Pop-up A pop-up appears when user tries to navigate away from the site with items in their shopping cart.
    Home Page Top Sellers Feature Products A static listing of to top selling products on the storefront's home page.
    Home Page Store Specials A static listing devoted to special offers on the storefront's home page.
    Interstitial Cross-sell page Cross-Sell An interstitial page appears between the Product Detail and Shopping Cart pages offering another product at a reduced price.
    Interstitial Up-sell page Up-Sell An interstitial page appears between the Product Detail and Shopping Cart pages offering a better version of the product at a reduced price. If the shopper chooses the up-sell the original product is replaced.

  11. Click Next. The Products and Pricing step appears.

  12. In the Pricing Information section, make sure the right Currency is selected. By default, your default locale’s currency is pre-selected. If you are limiting the offer to a specific locale, make sure the currency for that locale is selected.

  13. In the Products area, click Add Products to find the products you want to discount shipping costs for. The Find Product ID window appears.

    Click here to learn how to search and select product IDs.

    To find products, enter criteria in the fields and click Search. Enter as much about the product(s) as you can. Then, when the results appear, select the product(s) you want and click Add Selected Items.

    • In the Product ID, Name, and/or SKU fields, enter the ID, name, or of the product(s) you want to find.
    • From the or Status list, select the status of the product(s) you want to find.
    • From the Filter list, select if you want to filter the results to include or exclude certain types of products.
    • From the Catalog list, select the catalog with the product(s) you want to find.

  14. In the Discount Information area, select one of the Discount Options for the products you just added. You can choose to use the same discount for all products you have added, or define a different discount for each individual product.

    • To have the same discount for all products, select the Apply Discount to All Products option. Then, select a Discount Type from the options that appear below the Discount options.
    • To have a different discount for each product, select the Apply Individual Discounts to Products option. Then, in the table for each product listed, select a Discount Type from the list and enter the Discount amount in the field provided.

  15. If needed, change the Display Order for the products by entering a display orderDetermines the order in which variations will appear on your store, or the order in which products appear within an offer. value in the field provided. For example, a display order of "1" means the product appears first.

  16. Click Next. The Review step appears.

  17. On the Review page, make sure the offer was created correctly. If you need to change options or settings, click Previous to move back through the steps of the Offer Wizard.

    When you are done and ready to save the offer, click Finish and the following message appears. Click either Yes or No. Your offer has been created and you are returned to the Offers list page.
