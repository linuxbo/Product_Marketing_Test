Download

The Download fulfillment option is used when you have a product that you want shoppers to download upon purchase. This fulfillment method is usually used with digital products such as software, games, and so on.

Download Attributes

The following table outlines the attributes that can be set for products when the download fulfillment method is used.

Attribute

Definition

Download Fulfillment

Indicates whether Digital River or a third‑party will provide the download fulfillment for the product. If you select Digital River you will have to set the other options listed in this table.

Download File

The file(s) you want shoppers to download upon purchase. You can include as many files as needed but if

Download Protocol

Internet protocol used to transfer the file to the shopper’s computer.

Time Frame

How long after purchase you want to allow shoppers to download the product. Keep in mind that shoppers who purchase EDS will have additional time to download the product.

Number of Attempts

Number of attempts allowed per shopper to download the product.

Number of Attempted IP Addresses

Number of IP addresses that are allowed to attempt a download of the product.
