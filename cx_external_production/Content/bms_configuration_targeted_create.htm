Creating a Targeted Backup Media Configuration

Backup Media Configurations are the artwork and setup you want to be used with your Backup Media (Backup CD/DVD) orders. A targeted Backup Media configuration is a configuration that is only used when a specific product is purchased. For more about targeted configurations, see About Backup Media.

Note: The artwork you use for the mailer, disc image, and installation menu must meet specific guidelines. See the Mailer Design Guidelines and the CD and DVD Face Design Guidelines for information about how your artwork should be set up for a configuration.

To create a targeted Backup Media configuration:

Precondition: Target Segments must be enabled in Site Settings. Click here for more information about enabling this option.

 

Role Needed: Backup Media ManagerA user role that provides access to create Backup Media configurations. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select Backup Media Services, and click Create Configuration. The General Settings step of the Create Backup Media Configuration wizard appears.
  2. In the Name field, enter a name for the configuration.
  3. In the Description field, enter a description for the configuration if needed. Descriptions are not seen by shoppers.
  4. In the Dates area, choose when you want the configuration to start and end. You can choose to start using the configuration immediately after it is deployed, or select a particular date. You can also choose to end the configuration on a specific date or have no end date set.
  5. In the Supported Locales area, select the locales you want the configuration to support and click Next. The locales you select here will determine what locales this particular configuration will be used for. Make sure you indicate which locale you want to be the default.
  6. In the Target Segments step, in the Segment area, select Specific Product(s). Then, when the Products area appears, click Add to search for and then select those products that you want to use this configuration. When you are done adding products to the configuration, click Next to move on to the next step of the wizard.
  7. If the Trialware step, click Trial Product to add a trial to the configuration. The Find Products window appears.
    1. In the Find Product window, search for and then add the trialware products you want to include and add it to the configuration.
    2. Continue adding trialware products to the configuration until it contains all the trialware you want to include. When you are you will return to the wizard.

Note: Trials are added to the Backup Media disc as space permits.

  1. Click Next. The Artwork step of the Create Backup Media Configuration wizard appears. In this step, you will define the artwork you want used for the disc face and case.
  2. In the Artwork area you will see four tabs where you can upload or change the artwork files to be used for that item in the configuration:
    1. Disc – Artwork you want to be used to create the art on the top of the disc.
    2. US Mailer – Artwork you to be used for the mailers when the disc is shipped within the United States. This is the mailer design that will be used when the disc is shipped within the United States.
    3. EU Mailer – Artwork you want to be used for the European mailers. This is the mailer that will be used when the disc is shipped to countries in the EU.
    4. Install Menu – Shows a mockup of the installation menu shoppers will see when they run the installation. Click on the header or footer images on this tab to select a new image for the installation menu.
  3. On the Disc tab, click Select Artwork. In the window that appears, select the artwork you want to be used or upload the new artwork file and add it to the configuration.
  4. On the US Mailer and EU Mailer tabs, repeat the step above to upload a new file or select an existing file for the mailers. You may or may not have to create both mailers, depending on your store and what locales this configuration supports.
  5. When you are done uploading and selecting artwork files, click Next. The Review step of the Create Backup Media Configuration wizard appears.
  6. Review the configuration you have created. If you are satisfied with the layout, click Finish. The configuration is saved and you are returned to the Backup Media Configurations list page.
