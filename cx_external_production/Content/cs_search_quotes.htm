Searching Quotes

You can search for quotes by quote ID, start date or end date, or customer information (first or last name and email address). The search results are presented on the Quotes page. This page lists all the quotes in the search results and from here you can view an order, including the customer and product details for the quote.

To search quotes:

  1. On the Customer Service menu, select Search, and click Search Quotes.
  2. Enter your search criteria in the fields. You must enter criteria in at least one field. When entering search terms, you can use either uppercase or lowercase letters. Searches are not case sensitive. All fields search for an exact match with the exception of the first name field.
    • Quote ID – The system generated ID for the quote.
    • First Name – The quoted customer's first name.
    • Last Name – The quoted customer's last name.
    • Email Address – The quoted customer's email address.
    • Start Date – The start date for the quote, or the day when the quote was done.
    • End Date – The end date for the quote, or the day that the quote expires.
  3. Click Search. The page refreshes and the Quotes page appears showing the search results.
  4. Click the Order Number for the quote you want to view. The Order Summary page appears. From this page you can view and edit additional information about the quote such as customer information, order information, and product information.
