Business Navigator: Operational Reports

Operational reports show you information about the current state of your demand orders (inventory levels and unfulfilled orders).

Report Name Description
Cancelled Orders Detailed transaction report of cancelled orders.
Gifting Cancellation Reports Detailed transaction report of cancelled orders which were gifted.
Inventory Fulfillment Listing report of inventory available.
POP Stock Availability Report Listing report of inventory available by offer.
Unfulfilled Orders Detailed transaction report regarding orders which are yet to be fulfilled.
