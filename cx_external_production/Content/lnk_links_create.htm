Creating Links

Links are created using a wizard, where you define the type of link you want to create and the behavior it should have. When a shopper clicks a link, they can be taken to a specific page on your store, another Web site, or trigger certain actions such as adding a product to the Shopping Cart.

  1. On the Storefront menu, select Links and Parameters, and click Create Link. The Create Link page appears, showing the Type step.
  2. In the Type step, select the type of link you want to create. The type you select determines how the link functions and the options you set for the link in later steps. See Types of Links You Can Create for more about the type step.

Note: Once a link has been created and saved, you cannot change the link type.

  1. Click Next and the Parameters page appears. On this page, select or enter the parameters for the link. See Link Parameters for more about the parameters step.
  2. Click Next and the Options page appears. On this page, select (or edit) the options for the link and click Finish. See Link Options for more about the options step.

Types of Links You Can Create

In the first step of the wizard you are asked to choose a link type. The table below defines the actions that occur with each type of link.

Type

Action When Clicked

Add specific product(s) directly to my site's shopping cart

One or more products are automatically added to the shopping cart.

Go to a Shopper-Defined Bundle page

Shopper is taken to a interstitial page where they can choose from a list of products to create their own bundle.

Go to a specific product page

Shopper is taken to a specific product page.

Go to a specific category page

Shopper is taken to a specific category page.

Go to my store's home page

Shopper is taken to your storeâ€™s home page.

With this type of link you can select a theme, locale, currency, program, and offer to further define how the link operates.

Custom Page

Shopper is taken to a specific custom page you have created and uploaded to your store.

Go to a static Digital River hosted page

Shopper is taken to a page on a Digital River-hosted page.

Go to an external web page

Shopper is taken to a specific Web page not part of a Digital River hosted store or your store.

Other

You select the action you want the link to take when it is clicked.

Link Parameters

In the second step of the wizard you are asked to define the link "Parameters." The parameters define what type of page the link triggers, as well as what theme, locale, offer, and so on are used. The options that appear in the "Parameters" step are determined by the type of link you chose in the first step of the wizard.

The following table explains every possible parameter that can appear in the wizard. Keep in mind that you will only see those parameters that are relevant to the type of link you are creating.

Parameter Name

Definition

Name

Name for the link. Names do not appear on your store.

Description

Description for the link. Descriptions do not appear on your store.

Product ID

The product(s) being added to the Shopping Cart or viewed. You can enter a single product ID, a parent product ID, or the variation ID.

This field only appears when creating a link to automatically add products to the shopping cart or launch product pages.

Quantity

Quantity of product(s) you want to add to the Shopping Cart when the link is clicked.

This field only appears when creating a link that automatically adds products to the shopping cart.

Category ID

The category page you want to display when the link is clicked.

This field only appears when the link goes to a specific category page.

Parent Category ID

Parent category ID for the link.

Page Name

Custom page you want the link to open. This may already be selected if you came from a custom page to create the link.

Page Content Type

Layout of the page that is used to display the link.

  • Content with Shopper Framework puts the custom page into the main section of a regular store page.
  • Content alone creates a page with no navigation or branding. This is useful for offers where you want to limit the activity of shoppers.
  • Content with Theme appears only if you have set up a theme (see Using Themes to Apply Styles to Pages for more information). This field only appears if you are linking to custom pages.

Static URL

Digital River-hosted store you want the link to open. This field only appears when creating a static Digital River-hosted page.

External URL

External store you want the link to open.

Action

Action you want an "Other" type link to take when clicked.

Theme

The theme you want to load when the link is clicked. The theme you want must already be created and deployed within Store Builder themes.

Locale

The locale you want to load when the link is clicked. The locale you select must be supported on your store or the link will load the default locale for your store.

Currency

Currency in which the link displays. This must match the locale selected or the link will not work.

Program ID

The program you want to use to track traffic for the link. Programs track shopper traffic coming into a store from an external location, such as a Google keyword. When a URL with a program ID is clicked, the system logs the click. This data is used to determine click and buy ratios in reports.

For more information about programs, see About Tracking Programs.

Offer ID

The offer you want the link to trigger when it is clicked. If needed, offer IDs can be combined to trigger multiple discounts.

Parameter List

If your store uses unique name/value pairs (parameters), add them to the link here.

Link Options

The third step of the wizard prompts you to define the "Options" for the link. The options control how the link behaves, how it is created, or whether it is saved in the Command Console for future use. Link options also control whether the link expires and what happens when an expired link is clicked.

The list below outlines the options you can set during the third step of the wizard.

  • Hide Offer ID, Hide Program ID, Email Friendly URL, and Less than Eighty Characters â€“ Indicates you want to hide the offer and program IDs from the final link, as well as make the link URL shorter and "friendly" for email and Google purposes. If you select the "Expires" check box, additional fields appear so you can enter the expiration date and the URL where you want visitors to be taken when the link expires.
