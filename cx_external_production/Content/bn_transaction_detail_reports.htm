Business Navigator: Transaction Detail Reports

Transaction Detail reports provide detailed information about transactions on your site.

Report Name Description
Gifting Report Detailed report showing gifting orders.
Purchase Plan Shopper Detailed report showing orders grouped at the Purchase Plan level.
Transaction Accounting Detail Detailed report showing costed order information.
Transaction Customer Detail Detailed report showing order and customer information.
