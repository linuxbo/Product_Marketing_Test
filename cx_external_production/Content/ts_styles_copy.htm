Copying Styles

You can make a copy of a style and use that copy to create a whole new style for your store. This is especially helpful when you want to create a new style that is very similar to an existing style you have already created.

Note: Themes and styles are not available for stores that use the Headwater template. This means that if your store uses the Headwater template, you will not see the themes and styles options in the Store Builder menu as these Help topics indicate. To find out what template your store uses, contact your CMM.

When you copy a style all properties of the style are copied and can be changed once the copy is complete. If your company has multiple sites, you can choose to copy the style to another store/site. When you copy to another site you can choose if you want any files used in the copied style (such as button images) to override files of the same name in the other site/store.

When a style is copied, the new copy will have the words "Copy of" before the name to help you identify the copy from the original. You can then edit the style and change the or the properties of the style.

To copy a style:

Role Needed: Content ManagerA user role that provides access to various Store Builder features including themes and styles, liinks, files, compare pages, and Backup Media configurations. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, and click Styles.
  2. Select the style(s) you want to copy and click Copy.


    • If you have access to only one site/store in your company hierarchy, the style is copied on the current site and the page refreshes to show the copied style in the list.
    • If you have access to more than one site within your company, the Copy Styles to Another Site window appears. From the To This Site list, select the site to which you want to copy the style(s). Select the Allow Overwrite of Copied Files check box if you want future copies of the style to overwrite the copy you are currently making. This option is recommended if you plan on making changes to the style and copying the changed style to other sites.


  3. When you are done, click Save.
