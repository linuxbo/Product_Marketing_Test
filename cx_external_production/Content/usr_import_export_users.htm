Importing and Exporting Users

You can use the Import and Export feature to create new users or edit existing users of your company and site(s).

To create new users you first download a blank spreadsheet template. Then, you enter information about the user you want to create in the spreadsheet using the information and instructions founds in the spreadsheet template. Once you are done defining the users you want to create in the spreadsheet, you import the spreadsheet into the Command Console and the users are created.

To edit existing users, you must export the data for those users you want to edit. Once you have made the changes to the users in the spreadsheet, you import the spreadsheet into the Command Console and the changes are made.

Exporting All Users or a Blank Spreadsheet Template

To export a blank spreadsheet template or all users:

Role Needed: User ManagerA user role that allows the user to create and manage users, including assigning user roles to users. Search this file for the "user role definitions" topic for more information.
  1. On the Administration menu, select Users, and click Import and Export.
  2. In the Export Users area, for the Export Option, select what you want to export:
    • Select Blank Template if you want a blank template so you can create new users.
    • Select All Users if you want to export data for all the users created for your company so you can edit the information or roles for one or more users.
  3. Click Export. When prompted, save the file on your computer. Do not rename the file if you are going to import the spreadsheet back into the Command Console.

Importing Spreadsheets

When you've added new users or modified the existing user data, you can import the new spreadsheet file so that the changes are saved in the Command Console.

To import a spreadsheet with new or changed users:

Role Needed: User ManagerA user role that allows the user to create and manage users, including assigning user roles to users. Search this file for the "user role definitions" topic for more information.
  1. On the Administration menu, select Users, and click Import and Export.
  2. In the Import Users area, click Browse next to the Import File field.
  3. Locate the file you want to import and click Open.
  4. Click Import. A message appears when the import is complete.
