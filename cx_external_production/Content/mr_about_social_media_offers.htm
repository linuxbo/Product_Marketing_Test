Offers and Social Media

After you create and deploy an offer you can post the offer to your Facebook and/or Twitter accounts using SocialStream. This option will only show up if you have at least one Facebook or Twitter account set up with SocialStream, and if the offer is deployed and not part of a Site Optimizer test.

When you deploy an offer from the Offers list page, you will see an option to post the offer using SocialStream.

If you click the link to post the offer to SocialStream, the Post Offer window will appear. In that window you can select the SocialStream account(s) you want to post to, enter the content of your post, and choose when you want the post to appear. If needed, you can click Insert Tracking Link to create a tracking link to monitor the success of the post.

For more information about creating posts with SocialStream, see Creating a Post.
