Using the Site Trends App

The Site Trends app is fairly basic and easy to use. Most of the time you can tap on a chart of graph to expand the view or see more detail. Switching from landscape to portrait views may also change the detail of the view.

Note: If you want to know more about how the KPIs are defined and calculated, see Site Trends Key Performance Indicator (KPI) Definitions.

To view or change your configuration:

  • Tap the top left icon to edit the configuration. This is where you enter your security information such as User Name and Password.

To reload the latest data from the server:

  • Tap the top right icon to reload the latest data from the server. Data is not automatically loaded when Site Trends is launched.

To change view orientation:

  • Rotate the phone to switch between landscape and portrait view.
  • On small displays in landscape view, charts will use the full screen.

When viewing Site Trends:

  • Tap a KPI in the list to show the respective chart. An example of a the chart for Demand Sales is shown below. Once you are viewing the chart, tap on the chart to highlight individual data points and see their value.
