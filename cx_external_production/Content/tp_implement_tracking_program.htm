Implementing a Tracking Program

Creating a tracking program ID is easy and takes just a couple minutes. But creating the tracking program ID is only the first step to implementing a tracking program on your store. After you create a tracking program you must use that tracking program ID– either on your store or in a link– for the tracking program to be truly implemented.

The following outlines the steps you should take to implement a tracking program on your store. See the pages indicated to learn more about performing each step.

  1. Create the tracking program. The first step to implementing a tracking program on your store is to create the program itself. See the information in Creating Tracking Programs to learn more about how to create a tracking program.
  2. Create the link. After you create a tracking program you must create the link that will be tracked by the program. When shoppers click the link, the sales and activity resulting from the link will be tracked. This allows you to see how a tracking program performs and compares to others. When creating the link, we recommend that you select the "hide" option so that the offer ID and tracking program ID are hidden in the link, and the link itself is made "email friendly" and is under 80 characters. For more information about creating an using links, see About Links.
  3. Use the link. The final step to implementing a tracking program is to use the link behind a banner or button, or include it in an email campaign. Activity resulting from clicks of those links or buttons will be tracked with the tracking program and can be seen in a tracking program report.
