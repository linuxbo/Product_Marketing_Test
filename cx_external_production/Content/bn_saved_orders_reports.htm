Business Navigator: Saved Orders Reports

Saved Order Reports show information about all orders that use the estimates feature. To learn more about the estimates feature, see About Estimates or Configuring Site Settings: Request Estimate.

Report Name Description
Open Estimate To Order Conversion - Line Item Level - Standard Detailed report showing orders which originated from estimates, by line item.
Open Estimate To Order Conversion - Order Level - Standard Detailed report showing orders which originated from estimates, by order level.
Open Estimates - Standard Detailed report showing estimates.
