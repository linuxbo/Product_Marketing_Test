Localizing Your Catalog and Categories

The Product Categories page shows your catalog structure, including the categories and products within each category and locale. If your site supports multiple locales you may want to change your catalog and categories to support them. Changing the catalog structure to accommodate the differences between locales is called "localizing the catalog."

Localizing a catalog allows you to customize the appearance of the storefront to accommodate the specific interests and needs of shoppers in a given locale. You can localize your catalog by creating locale-specific categories and then editing the category details (such as the name) to accommodate the locale you are supporting.

For example, if your site sells DVDs you could create a special locale-specific category for your Japan locale called "Anime". Only shoppers in the Japan locale would see the Anime category and be able to purchase products within that category.

Creating locale-specific categories is the same as creating regular categories for your catalog except that you select the locale you are supporting from the Locale list in the Catalog Tree before you create the category.

To create locale-specific categories:

Role Needed: Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information.
  1. On the Catalog menu, select Catalogs and Categories, and click Manage Categories.
  2. From the Locale list, select the locale for which you want to customize your categories and catalog. When you choose a new locale, the page refreshes and the category structure for that locale appears.
  3. In the Catalogs list, select the catalog (by default the first catalog in the list is selected) you want to localize by clicking on the catalog name.

  1. To create a new category for locale-specific products:
    1. Click the category or catalog where you want to create the new category and click Create Category. The Create Category page appears.
    2. Enter the category details as you normally would. Make sure you enter the category details information as you want it to appear for the selected locale. For example, the Site Category Name of the category should be in the language of the locale you are supporting. See Creating and Editing Categories more specific information about creating categories in the Catalog Tree.
  2. When you are done, click Save. The page refreshes and the new category is created. Repeat these steps to create as many locale-specific categories as needed. Once the categories are created you can create or copy products into the categories and localize the product as needed.
