Resetting Passwords

When you edit a user you can reset the user's password. When a password is reset, the user will get an email informing them of their new password.

To reset a user's password:

  1. On the Administration menu, select Users, and click Manage Users. The Users list page appears.
  2. Click on the User Name of the user you want to reset the password for. The Edit User page appears and the Details tab is shown by default.
  3. On the Details tab, click Expire Password. When you do this, the user's current password is expired and the next time the user logs on, they will have to enter a new password. The new password can be the same as the old password, that is, passwords can be reused.
  4. If you make any other changes to the user, click Save. You do not have to save if you just reset the user's password.
