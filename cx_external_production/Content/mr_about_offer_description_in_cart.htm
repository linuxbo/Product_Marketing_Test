About Offer Description in Cart

The Offer Description in Cart  is text that appears (in the cart) when an offer has been triggered. We recommend you enter text that promotes or highlights the offer. For example, "You saved $10 on this order with our special holiday promotion" or something similar.

  • If the offer is for a single product, by default the offer description appears below the product in the Shopping Cart. See example
  • If the offer is for an entire order, by default the offer description appears at the bottom of the Shopping Cart. See example

Setting Offer Description in Cart

The offer description in cart is set on the fourth step (the "Display Content" step) of the offer wizard.

If you want to use a token to dynamically enter text, click Available Tokens and then copy and paste the token you want to use. To learn more about the tokens functionality, see Using Tokens.

Click here to learn how to add a token.

To add a token:

  1. Click the Available Tokens link. The Tokens window appears.
  2. Click in the field of the token you want to use. Note how the contents of the entire token field is highlighted.

  3. Press CTRL+C on the keyboard to copy the token to the clipboard (you can only copy one token at a time) and click Done to close the window.
  4. Place the cursor where you want to insert the token you just copied. Then, press CTRL+V to paste the copied token code from the clipboard.
  5. Repeat the steps above to add additional tokens.
