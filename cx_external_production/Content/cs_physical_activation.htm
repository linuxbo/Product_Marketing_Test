SoftwarePassport Physical Activations

You can send a SoftwarePassport activation code to a shopper from within the Command Console.

To send a SoftwarePassport activation code to a shopper:

  1. On the Customer Service menu, select SoftwarePassport, and click Physical Activation. The Physical Activation page appears.
  2. In the Email Address field, enter the email address where the shoppers wants the key sent after it has been generated.
  3. In the Activation ID field, enter the activation ID that is seen on the Enter Key window that appear when the shopper attempts to launch or install the product. See examples.

  1. In the Serial Number field, enter the serial number for the product being activated.
  2. In the Hardware Fingerprint field, enter the hardware fingerprint that is seen on the Enter Key window that appear when the shopper attempts to launch or install the product. See the examples above.
  3. Click Send Code. An email with the code will be sent the email address you entered.
