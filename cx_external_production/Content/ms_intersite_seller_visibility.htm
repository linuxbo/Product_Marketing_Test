Managing Seller Visibility

You can control which product sellers can see your product in the marketStream Common Catalog. This limits who can add the product to their catalog and sell it on their site. You choose which companies participating in the marketStream program (see About marketStream Channel Intersite for more about the program) that you want to see the product, thus hiding it from everyone else.

Warning: Until you set seller visibility for a product, all sellers in the marketStream program can see the product and sell it on their site. If you have set visibility for one or more sellers you must remove this visibility to have all sellers see your product again. See Allowing All Sellers to See the Product below for more information.

When you allow a specific seller to see a product, they can see it in the marketStream Products list and add it to their catalog to sell it on their site. Any seller that you have not explicitly allowed to see your product will not be able to see it in the marketStream Products list. The product will not appear in the marketStream Products list for that seller and they will not be able to see it on their site.

Note: If you remove visibility for a product from a seller that has already added your product to their catalog, an email is sent to the email address on record for that company notifying them of the change and that the product will be removed from their catalog. The product is removed from the product seller's catalog in three (3) days to allow any outstanding transactions to complete.

To allow a seller to see a product:

Role Needed: Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information.
  1. First, locate the product you want to set visibility for (on the Catalog menu, select Products, and then click on the Internal Product Name in the list that appears).
  2. Scroll down to the Channel area and click Set Seller Visibility. The Set marketStream Visibility page appears.

  3. Click Add Seller to find the seller you want to be able to see this product. A new window opens where you can search for the company you want to allow to see this product in the Common Catalog and sell it on their site.

  4. In the new window that opened, enter criteria about the company you want to find in the fields provided and click Search. This is the marketStream Program seller you want to be able to see the product in the Common Catalog.
  5. From the list of search results, select the company/seller you want to be able to see and sell the product, and click Add Selected Seller. The window closes and you are returned to the
    Set marketStream Visibility page.

  6. Repeat steps 3-5 for every seller/company that you want to be able to see or sell your product. Click Apply as you search and select sellers to save the list and continue working on the page. When you are done adding sellers to the list of allowed sellers, click Save.

Allowing All Sellers to See the Product

Until you set seller visibility for a product, all sellers in the marketStream program can see the product and sell it on their site. This means that once you perform the steps above, some sellers who once could see your product will find the product removed from their catalog after the three day grace period.

You can allow all sellers to see the product in the Common Catalog by removing all sellers/companies from the list of those allowed to see the product. This resets the product to the default state where it is visible to all sellers in the marketStream Common Catalog.

To allow all sellers to see your product in the Common Catalog:

  • On the Set marketStream Visibility page, select all the sellers in the list and click Remove. Make sure you select all sellers or the visibility restrictions will still be in place for the product.
