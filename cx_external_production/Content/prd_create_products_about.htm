About Creating Products

Products are created using the Create Product wizard, which guides you through each step of the creation process. The fields and options you see in the wizard vary based on the type of product you are creating, the options and selections you have made in previous steps, and how the product template for your company has been configured (see About the Product Template for more information).

Tip: When you edit a product, the process is slightly different because the steps of the wizard are combined to one page, making it easier to move between locales, settings, and options. You click the setting you want to change from a three-pane editor and the fields used to define that setting appear on the page. See About the Product Editor for more about how to use the Product Editor to change existing products.

Before You Begin

We recommend you have certain information ready when you create a product. This will help ensure that your products are set up completely and behave as you want once they are deployed on your store. The following list outlines what you should have, at minimum, before creating a product to ensure the best possible setup.

  • List of serial numbers and/or unlock codes (if supplying your own). The list, a .txt file, must be one code per line (a hard return between codes).
  • Application file (if product is downloadable)
  • Product images (thumbnail and full image pictures)
  • Product descriptions (short and long)
