Uploading Custom Pages

You can upload custom pages to use as static HTML pages on your site. After you upload a page you can edit the HTML as needed (see Creating Custom Pages for more information).

To upload a custom page:

Role Needed: Page DesignerA user role that provides access to several Store Builder features incluidng links, files, and custom pages. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, and click Manage Custom Pages.
  2. Click on the folder in the Directories list on the left where you want to upload the custom page. You do not have to select a directory; you can upload custom pages to the root folder. If needed, you can create additional directories to organize your uploaded custom pages.
Click here to learn how to create a directory.

You can create directories (folders) to organize your uploaded files. You cannot create a folder in the root directory; you must select a folder in which to create the new folder. The main (root) folder in Files cannot contain files and is read-only.

To create a directory:

  1. If needed, select the directory in the list on the left in which you want to create the new directory.
  2. Click Create Directory in the taskbar. The Create Directory window appears.
  3. In the Directory Name field, enter a name for the folder and click Create. In the name you cannot use spaces or the characters in the table below.
| (pipe) @ (at sign) ‘ (single quotes)
; (semicolon) # (pound sign) “ (quotation marks)
~ (tilde) $ (dollar sign) [ ] (brackets)
+ (plus sign) % (percent) { } (braces)
= (equal sign) ^ (caret) ( ) (parentheses)
, (comma) & (ampersand) / (slash)
? (question mark) * (asterisk) \ (backslash)
! (exclamation point) < > (angle brackets) ` (accent grave)
  1. Click Upload File. The Upload File window appears.
  2. Click Browse, select the file that you want to upload, and click Upload.
