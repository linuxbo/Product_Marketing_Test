About Catalogs

Catalogs are used to organize products on your store. Catalogs consist of categories, in which products are created and saved. Catalogs are created on the site-level, and each site can have multiple catalogs but only one can be designated as -+the default catalog.

Catalogs are initially created for your site by Digital River when the site is launched but internal users can create additional catalogs for clients as needed. If you are a client (and you have the Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information. role) you can view and edit the details of an existing catalog (including changing the search service used for the catalog).

Creating Catalogs

Note: This task can only be done by Digital River internal users.
  1. On the Catalog menu, select Catalogs and Categories, and click Create Catalog. The Create Catalog page appears.
  2. In the Name and Display Name fields, enter the name(s) for the catalog you want to create. You can use the same name for both but keep in mind that the Display Name may be seen by shoppers on your site.

Note: A Catalog ID will be created for the catalog when it is created by the system. The ID will be used to identify the catalog in the system and cannot be changed.

  1. Select the Options for the catalog:
    • Select Display Catalog if you want this catalog to appear on your store. If the catalog is not set to display, any products you add to that catalog cannot be seen or purchased on your site/store.
    • Select Publish Catalog if you want to have the catalog available for all sites owned by your company.
  2. From the Search Service list, choose the search service for your site.
    • In memory with Lucene – The Lucene search engine is used but the index data is rebuilt for each search request. This is slower than using normal Lucene search, but the results are more accurate. This option is best used with sites with a relatively small number of products in the catalog.
    • In memory – The database is searched directly.
    • Lucene – The Lucene search engine is used. This is the fastest option, but there is a small chance that the indexed data could be out of sync with the database and so the search could return inaccurate results. However, the index is rebuilt nightly to alleviate this issue.
    • Lucene with Failover – The Lucene search engine is used. If it encounters any problems an In Memory search is done.
    • No Op - No search service is used. If this option is selected, no search results will be returned to a shopper when they search your store.
  3. Select the Enhanced Search check box if you want to use the enhanced search features on your store. See the topics in the Store Search book for more about how to set up and use the enhanced search features.
  4. In the Short Description and Long Description fields, enter the short and/or long description for the catalog you are creating. These are only used internally and can be helpful to identify what products will be in the catalog and so on.
  5. When you are done and ready to create the catalog, click Save. When the page refreshes you are taken to the Product Categories page and you can see the new catalog you created (if you chose to display it) in the Catalogs list at left.

Editing Catalogs

Role Needed: Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information.
  1. On the Catalog menu, select Catalogs and Categories, and click Manage Catalogs. The Catalogs page appears.
  2. Click on the catalog you want to edit from the list of catalogs on your site. The Edit Catalog page appears.
  3. From the Editing Locale list select the locale you want to edit.
  4. The Supported Locales list shows which locales the catalog can be used with. Select or remove locales from this list as needed.
  5. Edit the rest of the catalog details as needed. Use the information in Creating Catalogs above to help you determine how you need to edit the settings for the catalog.


  6. When you are done making changes to the catalog, click Save.
