Deleting Links

When you create a link, you can choose to save the link for future use. When you are done with a link you can delete it and remove it from the list on the Links page.

Note: When you delete a link it means that the link is no longer available in the system for you to copy or preview. This will not affect any pages you linked to in the link, or pages where you inserted the link into as long as the pages or products you created the link for remain.

To delete a link:

Role Needed: Content ManagerA user role that provides access to various Store Builder features including themes and styles, liinks, files, compare pages, and Backup Media configurations.   1. On the Storefront menu, select Links and Parameters, and click Manage Links. The Links page appears.
  2. Select the link you want to delete by clicking the check box next to that link and click Delete. When the page refreshes the list is updated and the link you deleted no longer appears.
