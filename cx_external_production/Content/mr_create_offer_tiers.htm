Creating Tiered Discounts

Creating tiers allows you to create a discount based upon the quantity purchased (of an individual product, products with variations, or products within category) or the purchase amount (tiers based on the purchase amount can only be created for order-level discount offers). Discounts can be a percentage off or an amount off of the price. How you set up tiers will differ slightly based upon the selected discount level for the offer.

If you create tiers for offers discounting individual products you will only be able to set up tiers based upon the quantity of the product purchased. If you create tiers for offers discounting all products, you can set up tiers based upon either the quantity purchased or the purchase amount.

Setting up Tiers for Individual Product Offers

Note: If you set up tiers for individual products in a bundle offer, the tiers only apply to the Parent products in the offer.

To set up a tier:

Role Needed: Merchandising ManagerA user role that provides access to various merchandising features, including offers and tracking programs. Search this file for the "user role definitions" topic for more information.
  1. For the Quantity Tier Ranges option, select whether you want quantity restrictions to be totaled across all line items in the order or for a single item.
    • Per Line Item – Used when you want the quantity of a single line item to be used when determining quantity discounts.
    • Across Line Items – Used when you want the quantity of all line items in the order to be combined when determining quantity discounts.
  2. From the Discount list, select the tier discount type.
    • Quantity Tier/Amount Off – Select this option if you want to set up tiers based upon the quantity and want to take an amount off the purchase price. For instance, if the selected currency is GBP, you may choose to take off 5 GBP for one of your tiers and 10 GBP off for the next. Do not use Amount Off discount methods for offers that are localized for two or more locales. Instead, use Percent Off Discount Methods only.
    • Quantity Tier/Percent Off – Select this option if you want to set up tiers based upon the quantity and want to take a percentage off the purchase price.
  3. In the From box, enter the starting value for the tier. This represents the lowest value for that tier.
  4. In the To box, enter the ending value for the tier. This represents the highest value for the tier.

Note: If this is the last tier, you can leave the To box empty. If the To box is left empty, the shopper will qualify for the discount specified for the tier when they purchase a number of products at or equal to the minimum quantity specified for this tier.

  1. In the Value box, enter the percentage off for that tier.
  2. Click Add Tiers to add that tier to your table. Click the -sign if you wish to remove this tier from the table. When you click Add Tiers the page refreshes. Scroll back down to the table to add another tier.
  3. Repeat the steps above for every tier you want to create.
  4. If you are done adding tiers, click Save to save your pricing information.

Setting up Tiers for All Product Offers

The discount pricing for an offer is editable on the Create Offer: Pricing page. You can access this page from the Offer Summary page. This is the final page displayed when you create a new offer. From the Offer Summary page, click Edit in the Pricing area to bring up the Pricing page for your offer.

From the Offer Summary page, you can click either the pricing link at the top of the page, or click Edit in the Pricing section to bring up the Pricing page for this offer.

On this page you can set the tiers you want to apply for your offer. These tiers are set up in the Discount Information  area.

To set up a tier:

  1. For the Quantity Tier Ranges option, select whether you want quantity restrictions to be totaled across all line items in the order or for a single item.
    • Per Line Item – Used when you want the quantity of a single line item to be used when determining quantity discounts.
    • Across Line Items – Used when you want the quantity of all line items in the order to be combined when determining quantity discounts.
  2. From the Discount list, select the type of tier you want to set up.

Note: Do not use Amount Off discount methods for offers that are localized for two or more locales. Instead, use Percent Off discount methods only.

    • Amount Tier/Amount Off – (Available with order-level discounts only.) Select this option if you want to set up tiers based upon the purchase price amount and want to take an amount off the purchase price. For instance, if the selected currency is GBP, you may choose to take 5 GBP for one of your tiers and 10 GBP off for the next. All tiered discounts based upon an amount must start at zero.
    • Amount Tier/Percent Off – (Available with order-level discounts only.) Select this option if you want to set up tiers based upon the purchase amount and want to take a percentage off the purchase price. All tiered discounts based upon an amount must start at zero.
    • Quantity Tier/Amount Off – Select this option if you want to set up tiers based upon the quantity purchased and want to take an amount off the purchase price. For instance, if the selected currency is GBP, you may choose to take 5 GBP for one of your tiers and 10 GBP off for the next.
    • Quantity Tier/Percent Off – Select this option if you want to set up tiers based upon the quantity purchased and want to take a percentage off the purchase price.
  1. In the From box enter the starting value for the tier.
  2. In the To box enter the ending value for the tier.
  3. In the Value box, enter either the percentage or amount you want to use for this tier.
  4. Click Add Tiers to add that tier to your table. When you click Add Tiers the page refreshes. Scroll back down to this table. If you wish to remove this tier, click the -sign.
  5. Repeat steps 3 through 6 for each tier you want to add.
  6. Click Save to save the tiers you have set up.
