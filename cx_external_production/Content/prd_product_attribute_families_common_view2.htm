Viewing Common Product Attributes

You can view the common product attributes that can be used by all stores. Common product attributes can be viewed but they cannot be changed or edited.

To view common attributes:

Role Needed: Family ManagerA user role that provides access to common and custom product attributes. Search this file for the "user role definitions" topic for more information.
  1. On the Administration menu, select System, and click Common Product Attributes. The Common Product Attributes page appears.
  2. To view the history for a common product attributes, select the attribute group you want to view history for and click View History. The Family Audit History page appears. On this page you can see the changes made to the family. In the From Date and To Date fields, enter the timeframe for which you want to view changes and click Search. To return to the Common Product Attributes page, click Done.
  3. To view more about the attributes, click on the Name of the attributes group you want to view. The Common Product Attributes Details page appears. On this page you can see the locales and individual attributes and values in the group. To return to the Common Product Attributes page, click Done.
