Physical on Demand

The Physical on Demand fulfillment method is one where a physical product (disc) is created when the shopper places the order.

Physical on Demand Attributes

The following table outlines the attributes that are set for products when the Physical on Demand fulfillment method is used.

Attribute

Definition

Case Type

Note: The Case Type option only appears with POD products fulfilled by CMPS.

Indicates the disc case you want to be used for the POD product.

There are four different case types. The type you select also determines the weight of the POD product, which is used to calculate shipping costs for the POD product.

  • Cardboard mailer: 3 oz
  • Slim DVD: 5 oz
  • Standard DVD: 5 oz
  • Double DVD: 6 oz

Disc Label Image

Image or graphic to be printed on the disc.

Disc Case Image

PDF file with the image or graphic you want printed on the disc case insert.

Disc Content Files

File(s) for the actual disc. Usually these files include an auto-run or installer so the shopper doesn’t have to do anything when they want to install the product.

Attribute

Definition

Product Fulfillers

Lists the fulfillers that are available to fulfill your POD orders. You can select multiple fulfillers and Digital River will choose the fulfiller closest to the shipping address for any one order.

Part Number

Number assigned to the POD product by the specific fulfiller. Used to track and manage orders and inventory.
