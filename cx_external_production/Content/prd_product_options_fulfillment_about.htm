About Fulfillment

There are a few different ways you can distribute your product to shoppers, also known as fulfillment. How you choose to fulfill your product will depend on the product itself and any fulfillment agreements you have with Digital River or third‑party companies.

You can only use one fulfillment method per product. If you want to use more than one fulfillment option for a product, you should create a multiple product and have variations for each fulfillment method you want to use. This setup is common with software publishers who offer a physical version of their boxed software and a downloadable version of the same version.

To learn more about the fulfillment options for a product, see one of the topics listed below:

  • About the Download Fulfillment Option
  • About the Physical Fulfillment Option
  • About the Physical on Demand (POD) Fulfillment Option

Fulfillment Acknowledgement

If you are doing your own fulfillment, you can use the Fulfillment Acknowledgement page to help you track and manage the fulfillment of your products. See Using Fulfillment Acknowledgement.
