Adding and Removing a Twitter Account

You can use SocialStream to post marketing offers and promotions to your Twitter stream.

Adding a New Twitter Account

If you do not have an existing Twitter account, you can create one from within SocialStream and then add it.

Note: You cannot be logged in to Twitter when you add a Twitter account to SocialStream.

To add a new Twitter account:

Role Needed: SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select SocialStream, and click Dashboard. The Get Started with SocialStream page will appear.
  2. Click Sign up!within the Twitter area. A window will appear indicating that you will be directed to Twitter to sign up for an account.
  3. Click OK to continue. Twitter will appear in a new window.
  4. Follow the steps to set up a Twitter account, then return to the Command Console and follow the steps to add an existing Twitter account to SocialStream.

Adding an Existing Twitter Account

If you have an existing Twitter account, you can add it to SocialStream.

Note: You cannot be logged in to Twitter when you add a Twitter account to SocialStream.

To add an existing Twitter account:

Role Needed: SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select SocialStream, and click Dashboard. The Get Started with SocialStream page will appear.
  2. Click Add Twitter Account. The Accounts tab will appear in a new window, indicating that if you continue you will be directed to Twitter.
  3. Click Continue. You will be directed to Twitter and asked to allow SocialStream access.
  4. Click Allow to add your Twitter account to SocialStream. You will be directed back to a Command Console window displaying the Accounts tab.
  5. Click Continue. A message will appear indicating that your Facebook account has been verified and saved.
  6. Click Done to return to the SocialStream page.

Removing a Twitter Account

To remove a Twitter account:

Role Needed: SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select SocialStream, and click Dashboard.
  2. On the task bar, click Settings. The Accounts tab will appear in the SocialStream Settings window.
  3. Select a Twitter account from the list.
  4. Click Remove. A message appears warning you that removing the account will affect your live store. Follow the instructions in the message to approve or deny you want to make this change. If you approve the change, another message will appear indicating that you should also remove Digital River SocialStream from your Twitter account.
