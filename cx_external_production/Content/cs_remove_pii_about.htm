About Removing Personal Identifiable Information (PII)

The Remove Personal Info feature allows Customer Service Representatives to remove personal identifiable information (PII) from the system.

What is PII?

PII is any piece of information which could potentially be used to uniquely identify, contact, or locate a single person.

How Shoppers Request Removal

To have their personal information removed, shoppers must send a request to privacy@digitalriver.com and include the order ID and some personal information to verify their identity.

How PII is Removed

When a CSR clicks Remove Personal Info while viewing an order, the shopper's PII on that order is removed from the system. When a CSR is viewing an order with PII removed, the words "removed per shopper request" will appear to indicate the shopper has requested their PII be removed.

If the same shopper has a shopper account with the store (authenticated shopping) and has placed additional orders on the same site, the PII for those orders will also be removed. If the shopper doesn't have an account (anonymous shopping) with the site or has placed orders on other sites owned by the same company, only the PII on the current order will be removed.

What Information is Removed?

When a CSR clicks Remove Personal Info  while viewing an order, the following information (if present) will be deleted:

  • First and Last Name
  • Phonetic Spelling of First and Last Name
  • Login Name (for stores with authenticated shopper accounts)
  • Employer Name and Employer Department
  • Email Address(es)
  • Date of Birth
  • Phone Numbers (home, work, fax, etc.)

What Information is Not Removed?

If the shopper's address is within the United States, the following will not be removed:

  • City
  • State
  • ZIP Code

If the shopper's address is within Brazil, Canada, or India, the following will not be removed:

  • State
  • Country

If the shopper's address is in any other country, the following will not be removed:

  • Country
