Setting Colors and Fonts in Styles

You can change the default font and colors used with a style. Colors and fonts are likely the elements of your site you'd like the most control over because it includes hyperlinks, tables, and page background colors, amongst other elements.

Note: Themes and styles are not available for stores that use the Headwater template. This means that if your store uses the Headwater template, you will not see the themes and styles options in the Store Builder menu as these Help topics indicate. To find out what template your store uses, contact your CMM.

To change fonts and colors:

Role Needed: Content ManagerA user role that provides access to various Store Builder features including themes and styles, liinks, files, compare pages, and Backup Media configurations. Search this file for the "user role definitions" topic for more information.

Tip: The Colors & Fonts tab shows all the colors used in the style. A sample of the currently selected color appears next to each color value. If you want to modify a color you can either type the hexadecimal value into the field, or click the color picker to select a new color. Click here to see a sample:

  1. On the Storefront menu, select Store Builder, click Styles.
  2. Click on the Name of the style you want to edit.
  3. Click the Colors & Fonts tab.
  4. Enter (or edit) the name of the font family you want to use in the Font Family field at top. The font you choose here will be used for all heading and body text.

Tip: Click Apply as you enter information or make changes to save and continue working on the page.

  1. To change the heading size and color, make your changes in the Heading area. A heading is the text that signifies a new subject at the top or beginning of a section or paragraph. The following settings can be modified:
    • Size – The size of the heading text specified in pixels (px) or points (pt).
    • Color – The color of the heading text.
  2. To change the settings for non-heading text on the page (essentially everything that is not a heading, including body text and any notes or instructional text), make your changes in the Primary Text area. The following settings can be modified:
    • Size – The size of the heading text specified in pixels (px) or points (pt).
    • Color – The color of the heading text.
  3. To change how tables appear, make your changes in the Table Text & Colors area. Tables are generally used all over Web sites for an organized layout. For example, the shopping cart is typically a table. The following settings can be modified:
    • Header Size – The size of the table's header text specified in pixels (px) or points (pt).
    • Header Color – The color of the table's header text.
    • Body Size – The size of the table's body text specified in pixels (px) or points (pt).
    • Body Color – The color of the table's body text.
  4. To modify the link (hyperlink) colors for the style, make your changes in the Link Colors area. The following settings can be modified:
    • Link – The color of a link that has not been previously clicked (visited) and is not currently being hovered over by the shopper's cursor.
    • Hover – The color of a link when the shopper moves the cursor over it.
    • Visited – The color of a link that a shopper has clicked.
  5. To modify the background color of pages, body content, and tables, make your changes in the Background Colors area. The following settings can be modified:
    • Page – The color used for the page background.
    • Content – The color used for the content background.
    • Tables – The color used for the background of tables.
    • Odd Rows – The color used for background of the odd rows in tables.
    • Even Rows – The color used for background of the even rows in tables.
  6. When you are done, click Save to return to the main Styles list.
