Creating Links for Tracking Programs

After you create a tracking program, you must create a link for that tracking program or there will be no way for the program activity to be tracked. You can easily create a link for a tracking program using the Links feature.

Once you have a link with a tracking program ID, you can use that link in many ways such as with email campaigns, store links, banner ads purchased on other sites, and so on. When a link with a tracking program ID is clicked the activity resulting from that click are tracked, and the system use that information when generating one of several tracking program-related reports.

What Type of Link Should You Create

There are several different types of links you can create for a tracking program. The type you choose depends on what you want the tracking program to do, or how you want the offer being tracked by the program to function. Because links can be created for many different uses (for example, to take the shopper directly to a product page or take them to another Web site) there is no specific "right" link type to use.

Tracking Programs and Reports

There are several reports you can run to see the activity that resulted from a clicked link created for a tracking program. These reports show how many times a link was clicked, as well as information about sales and revenue generated from that click. The reports can also show information about the products and offers associated with a tracking program, and how tracking programs perform overall and over time.
