About Offer Triggers: Promotional URL

Offers that are triggered by a promotional URL require a shopper to click a link (URL) to redeem the offer.

Offer That Can Use Promotional URLs

The following types of offers can be used with the Promotional URL trigger:

  • All Products Bundle
  • Banner
  • Bundle
  • Buy M, Get N
  • Discount
  • All of the shipping offers

Restricting Promotional URL Usage

Because the promotional URL trigger type requires shoppers to click a link to trigger the offer, you can restrict the number of times a promotional URL offer can be redeemed. You can limit a shopper's individual use of a promotional URL offer (for example, a shopper can redeem the offer once) or limit the number of times the offer can redeemed on your store (for example, only 100 people can redeem the offer). Limiting the number of times a promotional URL offer can be redeemed will encourage shoppers to act fast to redeem the offer (if you inform them of the limited nature of the offer).

Creating Links for Promotional URL Offers

When you create an offer with the Promotional URL trigger, you should also create a link for the offer in Store Builder. Once you have a link you can copy and paste the link into an email, add it to your store, etc. Shoppers will click the link to trigger the offer. These types of offers are usually created to take a shopper directly to the shopping cart (with products already added) or to a product detail page so they can choose their delivery method, variation, and so on.

Using the Promotional URL Trigger to Add Products to Cart

If you are creating an offer that uses the Promotional URL trigger to automatically add a product to the shopping cart, you must use either an individual product ID, or a variation ID for products with variations. Do you not use the base product ID for products with variations or the link will take the shopper to the product details page for the base product (not the shopping cart, as you intend).
