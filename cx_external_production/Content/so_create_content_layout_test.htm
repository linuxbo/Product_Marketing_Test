Creating Content Layout Site Optimizer Tests

You can create a Site Optimizer test that compares one or more content layouts to see which content layout performs the best.

Note: Content layouts is only available if your store is using the Self Service (Rivulet) template. To find out what template your store uses, contact your Store Operations team.

To create a content layout Site Optimizer test:

Role Needed: Site Optimizer1.) A feature in the Command Console that allows users to conduct site-wide A/B testing on a live site. 2.) A user role that provides access to Site Optimizer. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select Site Optimizer Tests, and click create Site Optimizer Test. The Create Site Optimizer Test: Step 1 - Test Type page appears.
  2. On the Test Type page, select Content Layout for the Test Type and click Next.
  3. On the Information page:
    1. In the Test Name field, enter a name for the test. This will be shown in the reporting results for the test.
    2. In the Description field, enter a description for the test.
    3. From the Content Layout list, select the content layout you want to test.
    4. Enter a Start Date for the test. If this date is set in the future, the test will activate on your site on that date.
    5. Enter an End Date for the test. The end date is for data collection purposes only. Tests are not completed or retired until they are promoted, retired or ended manually.
    6. Click Next to move on the next step of the wizard.
  4. On the Levels page, click Add to add a level. Levels are the customizations you want to test in the content layout. After you add a level you should set up the level for the test using the steps below. Continue to add levels until you have the number of levels you intend to test.
    1. In the Name field that appears, enter a name for the level.
    2. In the Control column, select whether you want the level to be the Control level. Reporting will indicate that this level is the control to be compared with the other test levels. There can be only one control per test.
    3. In the Split fields, enter a percentage for each level. The split determines the percentage of visitors that will see the level and must add up to 100 across all test levels. The split is randomly assigned when the shopper first visits your storefront. The assigned test level is shown to the shopper during their entire session. Another test level is randomly assigned if the shopper completes an order or clears their browser cookies.
  1. After you have added all the levels you intend to test, click Next to move to the Markup step of the wizard where you will define the specific customizations you will test with each level.
  2. In the Markup area, you will see a text area showing the markup of the content layout you chose for the test. You can add this markup to each of your test levels and then customize that markup as desired to create the customizations you want to test.
    1. First, select a level from the list of levels you created and click Insert Markup. The markup you see below the levels list is then added to the level you selected.
    2. Now, customize the markup you just inserted as desired. Use the link provided on the page to open the help file associated with the Splash markup language that is used to define the content of the level.
    3. Continue adding markup to your levels and customizing it as needed to define the specific content you want to test. When you are done, click Next to continue to the next step of the wizard.
  3. On the Summary page, review the test you have created or edited.
    • If you are creating a new test, you can preview the test as it will appear on your site. Click Preview Test in the taskbar. A new window opens with the first test level activated. To switch from test level to test level, click Next and Previous in the Site Optimizer preview widget in the upper right-hand corner of your browser window. This will activate each test level in the test and display it in the window.

  4. When you are done, click Finish. If you set the test to begin upon activation, you are prompted to choose if you want to activate the test now. After you create the test you will want to view the test results, and promote test levels or end the test. See Viewing Test Results & Ending or Promoting Tests.
