Adding and Removing a Facebook Account

You can use SocialStream to post marketing offers and promotions to a Facebook page. SocialStream can only add posts to Facebook pages (also called "fan pages," "pages for businesses" or "public profiles"). SocialStream cannot post to personal profiles. See About SocialStream for more about using SocialStream with Facebook pages.

Adding a New Facebook Account

If you do not have an existing Facebook account, you can create one from within SocialStream and then add that account to SocialStream.

Note: You cannot be logged in to Facebook when you add a Facebook account to SocialStream.

To add a new Facebook account:

Role Needed: SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select SocialStream, and click Dashboard. The Get Started with SocialStream page appears.
  2. In the Facebook area, click Sign up!. The page refreshes and a message appears indicating that you will be directed to Facebook to sign up for an account.
  3. Click OK to continue. Facebook will appear in a new window.
  4. Follow the steps provided by Facebook to set up an account. When you are done, return to the Command Console and follow the steps in this file to add an existing Facebook account to SocialStream.

Adding an Existing Facebook Account

If you have an existing Facebook account, you can add it to SocialStream.

Note: You cannot be logged in to Facebook when you add a Facebook account to SocialStream.

To add an existing Facebook account: 

Role Needed: SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select SocialStream, and click Dashboard. The Get Started with SocialStream page appears.
  2. Click Add Facebook Account. The Accounts tab appears in a new window indicating that if you continue you will be directed to Facebook.
  3. Click Continue. You are directed to the Facebook login page.
  4. Log in to your Facebook account. Once you are logged in, you are directed back to a Command Console window with the Accounts tab.
  5. Click Continue. A message appears indicating that your Facebook account has been verified and saved.
  6. Click Done to return to the SocialStream page.

Removing a Facebook Account

To remove a Facebook account:

Role Needed: SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select SocialStream, and click Dashboard.
  2. On the taskbar, click Settings. The Accounts tab will appear in the SocialStream Settings window.
  3. Select a Facebook account from the list.
  4. Click Remove. A message appears asking you to confirm that you want to remove the Facebook account. If you want to continue, enter your username as prompted and click OK. Another message then appears reminding you to remove Digital River SocialStream from your Facebook account.
