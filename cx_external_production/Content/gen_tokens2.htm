Using Tokens

A token is a bit of code (similar to tags used in XML and HTML) that you can add to offers to add certain information or elements to the offer content. For example, you can insert a token that adds an "email this" link to an offer page so your shoppers can send the offer URL to a friend.

When you are creating an offer, you can use tokens in the Display Content step. The table below outlines the tokens that are available. If one of the tokens below is not available on your site, or if you want to inquire about getting a different token created for your site, contact your Store Operations team.

Token Name Description Token Tag
AJAX Mini Cart Inserts a mini-cart on the page. From the mini-cart, shoppers can view their cart contents and checkout. <% AJAX_MINI_CART %>
Cart Summary Inserts a cart summary on the page. The summary shows the number of items in the cart and the subtotal for all items. <% CART_SUMMARY %>
Content Inserts the content of the page. The Content token is only used in the <BODY> content for styles (see Editing HTML in Styles for more about where you use this token). <% CONTENT %>
Currency Selector Inserts a selector that allows the shopper to choose the display currency for the site. <% CURRENCY_SELECTOR %>
Email This Link Inserts an "Email This" link to allow a shopper to email the URL to an email address. <% EMAIL_THIS_LINK %>
Locale Selector Inserts a selector that allows the shopper to choose the locale for the site. <% LOCALE_SELECTOR %>
Product Finder Inserts the product finder token. <% PRODUCT_FINDER %>
Product Search Inserts a search field that allows shoppers to search products. <% PRODUCT_SEARCH %>
Related Products

Inserts the list of related products.

<% RELATED_PRODUCTS %>
Site ID Inserts the site ID for the site. <% SITE_ID %>
Theme ID Inserts the theme ID for the page. <% THEME_ID %>
