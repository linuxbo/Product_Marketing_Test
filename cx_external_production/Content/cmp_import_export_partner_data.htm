Import and Export Partner Data

You can import and export partner data using our Import & Export Partner Data bulk tool. Partner data is information saved about manufacturers, affiliates, merchants, and so on. The codes and IDs you enter in this spreadsheet are used, in part, to determine which orders were placed by partners.

To import partner data:

Role Needed: Company ManagerA user role that provies access to several company-level features. Search this file for the "user role definitions" topic for more information.
  1. On the Administration menu, select Company, and click Import and Export Partner Data.
  2. In the Import Partner Data area, next to the File field, click Browse.
  3. Select the spreadsheet file you want to import, then click Import. The file is imported. If there are any errors or warnings during the import, they will appear at the top of the page when the page refreshes. When the import completes, an email with the imported file attached will be sent to the email address saved in your user profile.  

To export partner data:

Role Needed: Company ManagerA user role that provies access to several company-level features. Search this file for the "user role definitions" topic for more information.
  1. On the Administration menu, select Company, and click Import and Export Partner Data.
  2. In the Export Partner Data area, for the Export Option, select what you want to export:
    • Select Blank Template to export a blank template in which you can enter partner data to import later.
    • Select All Partner Data to export a spreadsheet of all the partner data to edit.
  3. Click Export. You are prompted to open or save the file.
