Configuring Serial Numbers

When you enable serial numbers for a product you can choose to use a single serial number for each unit sold (every shopper receives the same number) or use a different serial number for every unit sold or order placed (every shopper or every completed sale receives a unique number).

To configure serial numbers:

Role Needed: Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information.
  1. Scroll down to the Digital Rights area and click Configure Digital Rights. The Configure Digital Rights page appears and by default the Serial Numbers tab is selected.
  2. Select Enable Serial Numbers.
  3. From the Generation Method list, select how you want to generate the serial number(s) for the product:
    • Select Manual List if you want to enter or import a list of serial numbers to be used by the product. Continue with the next step to define how you will generate and manage serial numbers.
    • Select Fixed Number if you want to enter a single serial number for all purchases of the product. When you select this option, the Fixed Number Input field appears. Enter the serial number you want to use for all purchases of the product in this field and click Save at the bottom of the page. You are done setting up the serial numbers for this product and are returned to the Create Product or Edit Product page for the product.
    • Select External if you have created a custom integration to deliver and revoke serial numbers. See About Custom Integrations for more information.
  4. From the Distribution list, select how you want serial numbers to be distributed with purchases of the product.
  5. In the Serial Numbers per Unit field, enter the number of serial numbers you want to use for each sale (individual unit) of the product.
  6. In the Notification Email field, type the email address where you want to be sent notification when the number of serial numbers for the product are running low.
  7. In the Initial Warning Number field, type the number at which you want to be notified that serial numbers are running low. When the available number of serial numbers for the product reaches this number, a notification is sent to the email address you provided in the step above.
  8. In the Notification Interval field, type the number of serial numbers that will be distributed after the initial warning is sent before another notification is sent. For example, if you enter "2" in this field, after an initial warning is sent, an additional notification will be sent after two serial numbers are used for the product, and every two numbers after that until the serial numbers are totally used.
  9. If the serial number and product are verified and sold from (Undefined variable: gc_cx_variables.PlatformName3), select the Local Purchases check box. If the serial number and product are verified and sold outside (Undefined variable: gc_cx_variables.PlatformName3), for example on the Trialware Network, or wrapped with Software Passport, select the Activation Service Required check box.
  10. When you are done enabling serial numbers, click Apply. Now you are ready to import or enter the serial numbers you want to use. See Importing and Exporting Digital Rights for more information.
