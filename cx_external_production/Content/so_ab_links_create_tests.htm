Creating A/B Link Tests

You can create simple A/B Link tests to compare offers or promotions on your site. When you create an A/B Link test, you can choose the type of link you want to use for the level and then define the options for the levels. These steps are very similar to the steps you take when creating a regular Site Optimizer test.

To create an A/B Link test:

Role Needed: Site Optimizer1.) A feature in the Command Console that allows users to conduct site-wide A/B testing on a live site. 2.) A user role that provides access to Site Optimizer. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select Site Optimizer Tests, and click Create Site Optimizer Test. The Create Site Optimizer Testpage appears, showing the Type step.
  2. For the Type option, select A/B Link.


  3. Click Next and the Information  step appears.
  4. In the Name field enter a name for the A/B test. If needed, enter a description for the link in the Description field. Descriptions are not required and cannot be seen by visitors to your site.
  5. In the Schedule area, define when the test should being and end.
  6. In the URL area, enter the URL where shoppers should be directed if they click a link created for the test after the test has ended.
  7. Click Next. The Levels step appears.
  8. Click Add to add the first level for the link. The Add Level  window appears.
  9. In the Level Name field, enter a name for the level. If needed, enter a description for the link in the Level Description field. Descriptions are not required and cannot be seen by visitors to your site.
  10. In the Type area, choose the type of link you want to create for the first level of the test.
Click here to see a summary of the types of links you can create for the levels.
  • Add specific product(s) directly to my store's shopping cart – When this link is clicked, one or more products are added to the shopping cart.
  • Go to a specific product page – When this link is clicked, the shopper is taken to a specific product page.
  • Go to a specific category page – When this link is clicked, the shopper is taken to a specific category page.
  • Go to my store's home page – When this link is clicked, the shopper is taken to your store’s home page. You can select the theme for this link, as well as some other options that affect the home page appearance.
  • Go to an external web page – When this link is clicked, the shopper is taken to a specific Web page not part of a Digital River hosted site or your site.
  • Custom Page – When this link is clicked, the shopper is taken to a specific custom page you have created or imported to your site.
  1. In the Parameters area, enter or choose the parameters for the level and click Save when done. The options that appear on the page (and in the example below) are determined by the link type you chose. All possible options are explained below but you will only see those options relevant to the type of link you are creating and your screen may not match what you see in the example below.
    • Name and Description – The name and description for the link. Names are required. Descriptions are not required and do not appear to visitors on your site.
    • Product ID – The products associated with the link (used when creating a link to automatically add products to the shopping cart or open product pages).

      Click here to learn how to search and select product IDs.

      To find products, enter criteria in the fields and click Search. Enter as much about the product(s) as you can. Then, when the results appear, select the product(s) you want and click Add Selected Items.

      • In the Product ID, Name, and/or SKU fields, enter the ID, name, or of the product(s) you want to find.
      • From the or Status list, select the status of the product(s) you want to find.
      • From the Filter list, select if you want to filter the results to include or exclude certain types of products.
      • From the Catalog list, select the catalog with the product(s) you want to find.

    • Quantity – The quantity of product(s) to add to the shopping cart when the link is clicked (used when creating a link to automatically add products to the shopping cart).
    • Category ID – The category to be opened by the link.

      Click here to learn how to search and select category IDs.

      The Find Category ID window appears when you click the magnifying glass icon next to Category ID field.

      To find a category, from the Catalog list, select the catalog with the categories you want to find and click Search. Then, when the results appear select the category you want and click Add Selected Items.

    • Parent Category ID – The parent category ID for the link.
    • Page Name – The custom page you want the link to open. This may already be selected if you came from a HTML Content page to create the link.
    • Page Content Type – The layout of the page that will show the link. Content with Shopper Framework puts the custom page into the main section of a regular storefront page. Content alone creates a page with no navigation or branding. This is useful for particular marketing offers where you want to limit the activity of shoppers. Content with Content Manager Theme appears only if you have set up a theme.
    • Theme – The theme associated with the page used by the link.
    • Locale – The locale used for the link.
    • Currency – The currency type used for the link. This must match the locale selected or the link will not work.
    • Tracking Program ID – The IDs for the tracking programs associated with the link.

      Click here to learn how to search and select tracking program IDs.

      The Find Tracking Program ID window appears when you click the magnifying glass icon next to Tracking Program ID field.

      To find tracking programs, enter criteria about the tracking program you want to find in the fields and click Search. When the results appear, select the tracking program you want and click Add Selected Items.

      • In the Tracking Program ID or Tracking Program Name fields, enter the ID or name of the tracking program you want to find.
      • From the or Media Type list, select the type of tracking program you want to find.

    • Offer ID – The IDs for the offers associated with the link.

      Click here to learn how to search and select offer IDs.

      The Find Offer ID window appears when you click the magnifying glass icon next to Offer ID field.

      To find offers, enter criteria in the fields and click Search. Enter as much about the offers as you can. Then, when the results appear, select the offers you want and click Add Selected Items.

      • In the Offer ID or Offer Name fields, enter the ID or name of the offers you want to find.
      • From the Offer Type or Status lists, select the type of offers or the status of the offers you want to find.
      • In the Parent Products and Child Products fields, enter the IDs for the parent or child products for the offers you want to find.

    • Static URL – The Digital River-hosted site you want the link to open.
    • External URL – The external site you want the link to open.
    • Action – The action you want an “Other” link to take when clicked.
  2. Repeat the steps above as needed to add additional levels to the test. After you have added all the levels you want, enter the Split for each level in the test and select a Control link. The Split determines the percentage of shoppers that are directed to the link (at random) when the link is clicked. When you are done with the levels, click Next on the Summary step.
  3.  On the Summary page, review the summary for the test. When you are done, click Finish.
    • To preview the test in a new window, click Preview Test.
    • To see the URL for a link, click the + next to the level name to see the URL. You can copy and paste this URL as needed into emails or other promotional materials.
