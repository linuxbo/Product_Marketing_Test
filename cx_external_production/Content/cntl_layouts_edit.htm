Creating and Editing Content Layouts

Content layouts are used to configure the design and layout for the pages used on your store. You can modify any of the layouts available to your store for a custom shopping experience.

Note: Content layouts is only available if your store is using the Self Service (Rivulet) template. To find out what template your store uses, contact your Store Operations team.

To edit a content layout:

Role Needed: Page DesignerA user role that provides access to several Store Builder features incluidng links, files, and custom pages. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, click Manage Content Layouts. The Content Layouts page appears.
  2. Click on the name of the layout you want to modify. See About Content Layouts for more about the pages that you can edit.
  3. In the Page Markup area, in the Markup field, make your changes to the code for the page. Keep in mind that the code you enter must be valid Splash tags. See the Splash Markup Language Help for more about how to use the Splash Markup Language to modify the page.
  4. When you are done making your code changes, click Save.
