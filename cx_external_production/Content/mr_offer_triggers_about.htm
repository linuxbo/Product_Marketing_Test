About Offer Triggers

The offer trigger determines how the offer is activated or redeemed by shoppers. Depending on the trigger, shoppers may have to click a link or enter a code during checkout to redeem the offer.

There are four types of triggers you can use with offers. Tor learn more about a specific trigger, click a link below:

  • Always Triggered
  • Promotional URL
  • Coupon Code
  • Site Test
