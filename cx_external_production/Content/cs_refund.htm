Initiating Refunds

The Refund page appears when you click Refund in the taskbar when viewing the Order Summarypage. From the Refund page you can initiate a refund on all or part of an order.

To initiate a refund, complete the fields on the Refund page using the information below as a guide. When you are done, click Submit at the bottom of the page. If needed, you can click Reset Values to clear the form and to start again.

When you submit the refund form, you may need to confirm that you want to continue the refund if certain conditions exist for the order. Follow the prompts that appear to complete the refund request.

When the refund is submitted, the Refund Instructions page appears showing the total amount refunded. See Viewing Refund Instructions for more information about what to do on this page.

The table below outlines the fields on the Refund page and explains how to use them.

Label Description
Billing Address The billing address used by the customer at the time the order was placed.
E-Mail Address The shopper's email address.
New E-Mail Address Type the shopper's new email address, if applicable. This also updates the order and user information.
Shopper Login The login ID the shopper used when placing the order. For orders that are not associated with an account, this is listed as anonymous.
Order Date The date and time the order was placed.
Order Number The identification number automatically assigned to the order.
Previous Refund Numbers If available, click these links for detailed information about previous refunds on this order.
Reason for This Refund Select a reason for the refund from the list box.
Comments You must add specific comments as appropriate concerning the refund.
Refund Type Depending on the level of refund being given, select Product Level, or Order Level. Customer service representatives should generally select product level unless they are refunding tax or shipping. See the information below about the specific fields in the areas that appear when you select one of these options.
Product Level Refund Lists information about the refundable items from the original order.
Order Level Refund Lists the order level refund types, for example, tax refunds, and shipping refunds. Only line items with a selectable check box are available for refund.
Tax The tax applied to the order
Shipping The shipping amount applied to the order. For shipped product orders only.
Subtotal Total of the tax and shipping total amounts for the order.
Total For supervisor use only.

The Product Level Refund area appears when you select Product Level for the Refund Type. This area lists information about the refundable items from the original order.

Click here to see a sample of the Product Level Refund area.

  • In the Refund field, enter the amount to be refunded for each product listed that requires a refund.
  • If a download product is present in the order, the Download URL Only check box may appear.

The Order Level Refund area appears when you select Order Level for the Refund Type.

This area lists the order level refund types, for example tax refunds and shipping refunds. Only line items with a selectable check box are available for refund.

Click here to see a sample of the Order Level Refund area.

  • To refund the entire order cost, select the check box in the Total row. When you select this option, all the other options are selected and values entered for Shipping and Subtotal.
  • To refund some or all of the Shipping or Subtotal cost of the order, enter the amount to be refunded in the appropriate field for that row.
  • If applicable, select the check box in the Tax row to refund the tax on the order. This option is selected by default when the Total check box is selected.
  • If "Not Available" appears in any row in this area it means that a refund cannot be processed for that line item.
