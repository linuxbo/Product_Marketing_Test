Creating Partner Plans

How you create a partner plan will differ based on how you want the partner plan to behave. This topic explains the wizard steps you must complete to create a new partner plan. For more specific information about the various settings and options you can set for a partner plan, see About Partner Plans.

Opening the Wizard

  1. On the Catalog menu, select Pricing & Plans, and click Create Partner Plan. The first step of the Create Partner Plan wizard appears.

Step 1: Settings

  1. In the Internal Name field, enter a name for the partner plan.
    • The name you enter here will appear to partners when they access the landing page if the partner plan is set to "public." If the partner plan is set to "private" your partners will not see this name.
  2. (Optional) In the Description field, enter a brief description for the partner plan.
    • Descriptions can be helpful if you create more than one partner plan and want to specify who the plan is intended for.

Options

  1. For the Availability option, select whether you want the partner plan to be public or private.
    • Public partner plans appear in a list from which partners select the plan they want to access. Private plans do not appear in the list and partners associated with private plans can only access the one private plan.
  2. For the Activation option, select whether you want partners to be automatically approved for this partner plan when they register, or whether you want to manually approve all partners.

Expiration

  1. For the Start Date option, select when you want the partner plant to start.
    • Partners who try to access the partner plan before it has started will get a message that the partner plan is not available.
  2. For the End Date option, select when you want the partner plan to end.
    • Partners who try to access the partner plan after it has ended will get a message that the partner plan is not available.

Plan Pricing

  1. From the Base Price List list, select the price list you want to be used for pricing of products in the partner plan.
  2. Click Next. The second step of the Create Partner Plan wizard appears.

Step 2: Sales Levels

  1. For the Type option, select whether you want to define the sales levels by the number of units sold or by the total amount of sales.
    • If you select a Type and then change it later, any sales levels you have defined will be deleted and you will have to create your sales levels again. This is because your sales levels are defined by the Type you select (units or sales).
  2. Click Add Level. The Add Sales Level window appears.
  3. Define the first sales level you want for the partner plan. To define a level, in the To field, enter the value that corresponds to the upper limit you want for the first level in your partner plan.
    •  For example, if your partner plan is unit-based and you want sales level for 1-5 units, 6-10 units, and 11+ units you would enter 5 in the To field when creating the first level, 10  when creating the second level, and then select Infinity  when creating the third level.
    • When you create the next level you simply enter the upper limit for the second sales level in your plan and repeat as often as needed to define the plan levels.
  4. If you chose to define the sales levels by sales, from the Currency list, select a currency for the level you are creating.
  5. For the Sales Level Discount option, select whether you want to provide a discount for the sales level you are creating. If you select Percent Off, enter the discount percentage in the field provided. If you select Amount Off, enter the discount value in each of each of the currency fields that appear.
  6. In the Name field, enter a name for the sales level.
  7. (Optional) From the Theme list, select the theme you want to be used with the sales level.
    • Selecting a theme is optional. If you do not select a theme, the default theme for the store will be used.
    • Using a theme with a sales level allows you to tailor the look and feel of the page your partners see to partners within that level.
  8. Review what you have defined and make sure the sales level is exactly as you want it. Once you save this level it cannot be edited and can only be deleted. When you are sure the sales level is exactly as you want, click Save to create the level you have defined.
  9. Repeat the steps to add as many additional levels as you want in your partner plan. Make sure your top sales level is set to "infinity." When you are done, click Next to continue to the third step of the Create Partner Plan wizard.

Step 3: Product Groups

  1. In the list of levels on the left, select the sales level for which you want to create a product group.
    • You should create at least one group for each sales level you have created, otherwise you will not be able to add products to that level (products must be added to groups in sales levels).
  2. Click Add Group. The Create a Group window appears.
  3. In the Name field, enter a name for the group.
  4. In the Pricing area, for the Group Discount option, select how you want to discount products that are in this group (you will add products in a later step).
    • Group discounts are always applied to purchases.
    • If there is a sales level discount and a group discount, the group discount will be used. If there is no group discount, the sales level discount will be used.
  5. Click Apply. You will not be able to add products to the group until you save the product group with a name and the group discount selected.
  6. In the Order Restrictions area, for the Maximum Quantity and Minimum Quantity options, indicate whether you want to place purchase restrictions on this product group.
  7. In the Products area, click Add. The Find Products window appears.
  8. Search for and then add the product(s) you want to be in the group you are creating. When you are done adding the products to the group, click Save in the Create a Group window.
    • You can add one product or many products, depending on how you are setting up your partner plan.
  9. Continue creating groups and adding products to groups for each sales level in your partner plan. When you are done creating groups and adding products, click Next. The fourth step of the Create Partner Plan wizard.

Step 4: Pricing Review

  1. In the Pricing Review area, in the list of sales levels and products groups on the left, select a sales level and then select a product group within that level.
  2. From the Currency list, select the currency you want to be used for pricing and discounts for the selected group.
    • The currency you select for a product group will be used to calculate the price for products in the group. If you do not select a currency, the default currency will be used.
  3. Repeat the steps above to assign a currency to every product group in every sales level in the partner plan.
  4. When you are done assigning currencies to the product groups in your partner plan, click Next. The final step of the Create Partner Plan wizard appears.

Step 5: Review

  1. Review the partner plan you have created. If anything about the partner plan is not how you want, click Previous to go back steps and change what you have defined. Keep in mind that you cannot edit sales levels (you must delete them and recreate them).
  2. When the partner plan is set up the way you want, click Finish. The partner plan is created and you are taken to the Manage Partner Plans list page.
