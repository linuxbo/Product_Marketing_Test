About Marketing Dashboards

The Marketing Dashboards show how your various marketing efforts are performing on your store. Each dashboard shows different metrics (called Key Performance Indicators, or KPIs) in various ways such as tables, charts, and graphs.

The following defines the different types of dashboards you can view:

  • Merchandising – Shows how your offers and tracking programs are performing.
  • Bid Management – Shows how your paid search programs are performing.
  • Revenue Scorecard – Shows information on sales and revenue for your store.
  • Email Marketing – Shows information on your email marketing campaigns and promotions.
  • Lifetime Value – Shows information about the lifetime value of shoppers on your store.

Roles to View Dashboards

Each dashboard has a unique role that you must have to view the dashboard. To learn more about the roles needed for dashboards, see User Role Definitions. Role information is also found in the topics that explain how to view the dashboards.

Dashboard Data Filters

Each dashboard has a set of fields and options at the top that are used to define the data shown in the dashboard. Not all dashboards use the same filters.

Site(s)

  • The site(s) you want to see in the dashboard.
  • If you have the role at the company level you will be able to select any sites for that company.
  • If you have the role at the site level, you will only be able to select those sites for which you have been given the role.

Time Frame

  • The period of time used to define the data you see in the dashboard.
  • Click here to learn more about how time frames are defined.

Currency

  • The currency used to show data in the dashboard.
  • By default all data is shown in USD but you can select one of several other currencies as needed.
  • Click here to learn more about the currencies that are available.

Locale

  • The locale you want to used in the dashboard.
  • The locales you can select are the locales supported by all of the sites you have selected. This means that if changing the sites you have selected may change the locales you can select.

Dashboard Elements

Dashboards use different elements to show you how a particular metric has performed over time. Understanding how to use or interact with these elements can help you get the most out of using the dashboards.

The following elements are used by some or all of the Marketing Dashboards:

  • Sparklines
  • Trending Arrows
