Using Import and Export to Make marketStream Channel Category Assignments

You can use the import and export tool to assign (or remove) categories to marketStream Channel products in bulk. This is done using a spreadsheet where you enter the category ID in a cell to add or remove the product to that category. You can always review the information and instructions in the spreadsheet file itself for more detailed help.

The first step to assigning categories is to export a blank template or a spreadsheet of marketStream Channel products with the current category assignments. Then, you edit the information in the template to assign products to categories or remove products from categories. Finally, you import the spreadsheet with the changes you have made. When the import completes, the products are updated.

Note: You may find it helpful to download the category resource tree for the site before you edit the spreadsheet. The category resource tree contains a listing of the category IDs for the site. See Exporting the Category Text File for more information.

To change category assignments using import and export:

Role Needed: Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information.
  1. On the Catalog menu, select Catalogs and Categories, and click Import and Export Categories. The Categories Import and Export page appears.
  2. Click marketStream Channel Product Categories.
  3. In the Export Categories area, from the Catalog list, select the catalog with the products you want to export.
  4. For the Export Option, select All Active marketStream Products.
  5. In the Job Name field, enter a name for the export job. This name will be used to identify the job in the Job Queue.
  6. Click Export. The export begins and you are taken to the Import and Export Job Queue.  
  7. From the queue you can view the progress of the job. When the job completes, click on the Name of the job in the queue. The Job Detail page appears.
  8. Click the link to download the spreadsheet file with the marketStream Channel product category associations. When prompted, open or save the spreadsheet file on your computer.
  1. In the spreadsheet file, make the category assignments as needed:
    • If you want to add a product to a category, enter that category ID in the Add to Category cell.
    • If you want to remove the product from a category, enter that category ID in the Remove from Category cell.
  2. When you are done editing the spreadsheet, go back to Categories Import and Export page and click marketStream Channel Product Categories again. Then, in the Import Categories area, in the Job Name field, enter a name for the import job of the spreadsheet changes and click Browse to locate the spreadsheet file with the changes you want to import.
  3. Click Import. The import job begins and you are taken to the Job Queue. From the queue you can monitor the progress of the job. When the job completes, the marketStream Channel products are updated to be assigned (or removed) from the categories you indicated in the spreadsheet.
