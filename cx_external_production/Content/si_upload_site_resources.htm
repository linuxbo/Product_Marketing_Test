Uploading Site Resources

You can upload a Zip® file of resource files, such as images or multimedia files to your store. Uploading your resource files using a Zip file saves you from having to upload each resource file individually.

You are limited to adding resources using one of four predefined top-level directories. Those directories include images, javascript, content, and multimedia. Below the top level directory, you can create and use any allowable file structure.

For example, within your Zip® file, below your directory named Images, you could have other directories that include different types of images, such as software, hardware, and books. Just remember that the top level directory must be named using one of the predefined directory names.

To upload resources:

Role Needed: File Upload ManagerA user role that provides access to upload and download resrouce files in the Site Implementation and adminstration template features. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Site Implementation, and click Upload Resources. The Upload Site Implementation Resource Zip page appears.
  2. Enter the Zip® file name in the File box or click Browse to navigate to the location of the Zip file. File names must be in the format "yourfilename.zip". Keep in mind that you cannot use spaces in the file names or directory names within the Zip® file.
  3. Click Upload. A message appears when the upload is complete.
