Creating Coupon Code Offers

Coupon code offers provide discounts on products or shipping costs when a shopper enters a coupon code during checkout.

This topic explains the settings specific to coupon code offers, including how to add or manage coupon codes after an offer is created. We recommend you use this topic in conjunction with the Creating Marketing Offers topic if you want to create a coupon code offer.

Selecting the Coupon Code Trigger

In order to create a coupon code offer, you must select the coupon code trigger in the first step of the Create Offer wizard. Keep in mind that the coupon code trigger can only be used with specific offer types.

Choosing How to Get Codes for Your Offer

In the second step (the Behavior step) you will see the Coupon Codes area where you select how you want to add/generate coupon codes for your offer.

  • Enter Coupon Codes – You type/enter the coupon code(s) you want to use (for example "25%off").
  • Generate Coupon Codes – The system generates a set of random coupon codes based on some parameters you define. System generated codes are always nine (9) digits long.
  • Range – The system generates a range of coupon codes. The codes can have a specific prefix and you can specify the first, and/or last number you want to use for the range.
  • Import Coupon Codes – You import a .txt file that contains coupon codes you have created or generated elsewhere.

Note: Technically there is no limit on the number of coupon codes you can associate with an offer, just a limit on the number of codes you can import or generate at any single time. If you need more codes for your offer you should perform multiple imports or generate multiple sets of codes.

Limiting Coupon Code Usage

After you select a generation method, choose whether you want to limit the use of individual coupon codes. This only limits the number of times each code can be used. If you want to limit the use of the offer itself (for example, how many times the offer can be redeemed on your store) use the Offer Controls.

  • Select the Limit Individual Coupon Usage check box to limit the number of times a single coupon can be used. Then enter the number of times each code can be used in the Coupon Limit field that appears.

Coupon Code POP

Offers that use the coupon code trigger must use the "PointOfSale_CouponEntryField" POP for the coupon code field to appear in the shopping cart. If the offer is not set up with this POP, the field where shoppers enter the code will not appear. If the coupon code POP is not available as an option when you are creating the offer it means that POP has not been associated with that offer type. See Configuring Offer Types for more information.

Adding Coupon Codes to the Offer

When you reach the last step of the wizard (the Review step) you will see an option at the bottom of the page to indicate whether you want to set up the coupon codes for the offer now or later.

We recommend you select Yes to add coupon codes immediately after you finish the wizard, and then click Finish to move to the step where you add coupon codes. (When you click Finish, you will be prompted to choose if you want to deploy the offer. We recommend you do not deploy the offer until after you have set up coupon codes.) If you choose not to add coupon codes now, when you return to the offer and view it, you can use the steps below to add coupon codes to the offer.

Once you finish the wizard (or view an offer with the coupon code trigger) you will see the Offer Details page. In the Behavior area, click Add. A window appears where you can add or generate coupon codes for the offer, depending on the generation method you chose.

Use the steps below to add coupon codes to the offer depending on your generation method. When you are done, click Done at the bottom of the page to complete the coupon code setup.

If you chose the "Enter Coupon Codes" generation method:

  • In the Codes field, enter the coupon codes that shoppers can use to redeem the offer and click Save.

Guidelines for creating your own coupon codes:

  • Codes must be less than or greater than nine characters (nine characters are reserved for system generated codes) but no more than 64 characters in length. Keep in mind that shoppers will need to enter the code to redeem the offer so we recommend you keep your codes as short as possible.
  • Use commas to separate individual coupon codes, but do not use commas in the coupon codes themselves. A comma in a coupon code is treated as a line break. For example, if you enter "123,456" it would be interpreted as two codes ("123" and "456"). Similarly, if a shopper entered "123,456" in the coupon code field it would be interpreted as two different codes ("123" and "456").
  • Do not use spaces to separate individual coupon codes, or within the codes themselves. We do not recommend using spaces in coupon codes because it may lead to errors when the shopper is copying and pasting the coupon code during checkout, or in how the coupon code will display in an email or social media post.
  • Coupon codes can use special characters such as ! or & or %. You can use any standard symbol (on an English/United States keyboard) in your coupon codes, with the exception of commas as mentioned above.

If you chose the "Generate Coupon Codes" generation method :

  • In the Quantity field, enter the number of codes you want the system to generate for the offer. Codes will be generated using an algorithm developed by Digital River and will be nine characters in length.
  • You can generate up to two million codes at any one time. We recommend you generate codes multiple times if you need more than two million codes for your offer.

If you selected "Range" generation method:

  1. In the Prefix field, enter the prefix that you want to appear at the beginning of each code. Prefixes can be letters and/or numbers only with no spaces or special symbols.
  2. In the Start field, enter the first number for the range. Your range of coupon codes will begin with this number.
  3. In the End field, enter the last number for the range. Your range of coupon codes will end with this number.
Example: Here is how a range of codes would look if you specified a Prefix of FreeShip, a Start of 10, and an End of 15: FreeShip10, FreeShip11, FreeShip12, FreeShip13, FreeShip14, FreeShip15

If you selected "Import Coupon Codes" generation method:

  • Click Browse to select the .txt file containing the coupon codes you want to import. Click here to see an example of an import file.
  • Import files are limited to one million codes. We have found that files with more than a million codes can have problems importing. We recommend you do multiple imports if you need more than one million coupon codes for your offer.
