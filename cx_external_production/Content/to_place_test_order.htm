Placing Test Orders

Test orders are used to test the buying experience for a shopper on your store. When you place a test order you can mimic certain aspects of the buying experience such as how and where shoppers enter billing and shipping information and the emails they receive when their order is complete.

Opening the Store Preview

Role Needed: Test Order ManagerA user role that allows the user to preview the store and place test orders. Search this file for the "user role definitions" topic for more information.
  1. In the icon well, hover over the Preview Store option.

  2. From the Locale list, select the locale you want to use to place the test order.
  3. For the Preview State option, select whether you want to view the design version or live version of your store. Most often test orders are done on the live version of a store, unless there are specific design changes you want to test.
  4. For the Test Order option, make sure On is selected. To change what is selected for this option just click on toggle switch and the option will change.

  5. Click Launch Preview. A new tab or window will open showing your store with the options you selected.

Adding Products to the Shopping Cart

  1. Navigate your store and add products to your shopping cart. If you want to test specific products, locate those products and add them to your shopping cart. Otherwise you can add any product(s) to your shopping cart if you are just testing the shopping experience overall.
  2. If you are also testing a merchandising offer, make sure you trigger the merchandising offer(s) you are testing by adding the specific product(s) to the shopping cart that the offer is for, or by applying the coupon code in the shopping cart.

Checking Out

  1. When you are ready to place the test order, begin the checkout process. Enter your billing and shipping information using the information below as a guide.

Important! To place a test order correctly, you must use the credit card and checkout information outlined below. Digital River has a test credit card number that indicates to the system that the order is not a real order and so some internal billing and fulfillment processes should be skipped during checkout.

  • Make sure you fill out all the required fields to create an account for the store (if applicable).
  • You can use a made-up address and phone number as these items are not relevant for a test order.
  • Make sure you use the email address associated with your account login if you chose to receive emails in the previous step. 
  • Use the following credit card number: 4444333322221111. Credit cards are the only payment method supported by test orders.
  • Use today’s date as the expiration date for the credit card.
  • You do not need to enter a card security code (also known as a CV2 code). \

Selecting Your Test Order Options

  1. Scroll to the bottom of the page. You will see a set of special fields that you can use to define exactly how you want the test order to behave.

Note: The fields shown above also appear when you preview an offer. See Previewing Offers for more about how to preview an offer.

  1. Make sure the Is the order a TestOrder check box is selected.
  2. Select the Send Emails check box if you want email notifications that are usually sent to shoppers when they place an order (such as an Order Confirmation email) sent to the email address you entered in a previous step. We recommend you select this check box every time you place a test order.
  3. By default, the Fraud, Denied Parties, and Export Controls options are set to be bypassed. These options test additional aspects of the order process such as the fraud review, the denied parties list check, and the export controls regulations.

    Bypassing these options will speed up your test but you may choose to test these aspects of your order as well. Digital River runs those tests internally each release so you most likely will never need to test those items but you can if needed.

Finishing Your Test Order

  1. Once you have set your test order options and everything is ready, complete the checkout process. The steps to complete checkout vary by site so just follow the prompts until you see the Thank You page. Once your order is submitted you will receive an Order Confirmation email (if you chose to receive emails for this test order) and your test order is complete.
