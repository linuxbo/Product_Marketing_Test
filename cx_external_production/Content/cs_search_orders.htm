Searching Orders

You can search for orders by order ID, first and/or last name of shopper, and other attributes such as email address or credit card number. After you search, you can view the order summary and view specifics about the order such as customer information and product information.

To search for orders:

  1. On the Customer Service menu, select Search, and click Search Orders. The Search Orders page appears.
  2. For the Search By option at top, select how you want to search for orders. When you select an option, the bottom of the page refreshes to show fields and options to define your search with that criteria.
    • Order Number
    • Email Address
    • Credit Card Number
    • Shopper Login ID
    • Return Number (RMA)
    • External User ID
    • Phone Number
    • First and Last Name
  3. If the Commerce Platform list appears, select the site/store where the order was placed. If you select a specific site you will only be able to search for orders using the Order Number.
  4. Enter criteria about the order(s) you want to find in the fields that appear on the page. The fields are not case-sensitive so you do not have to worry about the case of the text you enter.
    1. Depending on the Search By option you selected, the Include Test Orders and Include Incomplete Orders check boxes may appear. You can use these check boxes to have the search include test orders and incomplete orders in the results.
    2. The Partial Match check box may appear when you can have system treat any text you have entered as partial entries, not full complete entries. For example, if you select the Partial Match check box and enter "jo" into the First Name field, the search will find orders placed by any shopper whose name begins with the letters "jo". The Partial Match option is helpful for names and other wording that can have multiple spellings or formats.
  1. Click Search. If the search returns results, they appear on the Order List page.
    • Click the Order Number to open and view the order.
    • Click the Login ID to view the Shopper Details page for the shopper who placed the order.
