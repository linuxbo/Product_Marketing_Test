Viewing ePen Reports

The ePen (or "electronic penetration") report provides monthly accounting detail for commerce transactions. There are several different reports that make up the ePen report set, including reports detailing fees paid and commerce transactions on the site. The ePen report is automatically generated and available each month. If a site has no sales for the month, the report will not be generated. Historic reports can also be viewed or downloaded.

Note: To view ePen reports you must have the View ePen Report role.

Each individual report in the ePen report can be viewed with Microsoft® Excel but you will also need WinZip® to extract the individual reports from the compressed file in which they are delivered.

The ePen report is used by accounting and finance personnel to reconcile Digital River payments and transactions for the payment period. This report is typically not used for marketing or promotional tracking purposes. Clients interested in data about sales, product promotions, product performance, site traffic, and operational data should generate one of the 60+ reports available in the Online Reporting tool (see About Online Reports for more about this tool).

Important! Microsoft® Excel® users may experience problems with their Detail ePen reports if the report contains double-byte characters. See Installing the ePen Report Macro to view more about this issue and how to fix it.

To view an ePen report:

Role Needed: View ePen ReportA user role that provides the user with the ability to view and download epen reports. Search this file for the "user role definitions" topic for more information.
  1. On the Reports menu, click ePen Reports. The ePen Reports page appears.
  2. Click on the name of the report you want to view in the Available Files list. A window appears prompting you to open or save the Zip® file of the report. Choose a location on your computer to save the report.
