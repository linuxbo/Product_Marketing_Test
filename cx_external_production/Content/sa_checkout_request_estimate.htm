Configuring Site Settings: Request Estimate

The Request Estimate settings are used to provide something similar to a quote feature but without the price guarantee. When you enable this feature on your site, it adds a "Request Estimate" button to your shopping cart so that shoppers can get an estimate of the contents in their cart. Estimates do not guarantee the pricing of the items in the shopping cart, they only save the contents of the cart so the shopper can return to your store and place and order. To learn more about estimates, see About Estimates.

Who Can View or Edit This Setting

Your ability to view or edit a specific site setting varies as follows:

  • The ability to view or edit a setting depends on the template used by the currently selected site, the type of user you are (internal Digital River user, or client user), and the user roles you have.

About the Request Estimates Setting

The picture below shows the Request Estimate settings area of site settings. The information below the picture defines these settings and what happens if you set them or how to configure them for your store.

Request Estimate:

  • Select Enabled if you want to provide the estimates features on your store.

Order Limit:

  • Select Single if you want to limit the number of orders a shopper can place from an estimate to a single order.
  • Select Unlimited if you want to allow shoppers to place multiple orders form the same estimate.

Set Estimate Expiration:

  • Select how long you want an estimate to be valid after it is created. How long an estimate stays valid determines how long a shopper can access that estimate and place an order from that estimate.

Send Reminder Email:

  • Select whether you want a reminder email sent to a shopper that has requested an estimate but not yet placed an order from that estimate. If you choose to have reminder emails sent, you'll have to indicate how long after an estimate is created the reminder email is sent. Please note that only one reminder can be sent.

Display Contact Me Checkbox:

  • Select whether you want a check box appear when shoppers are requesting an estimate which gives permission for customer service or sales to contact the shopper (via phone or email) regarding their estimate.

Company Representative:

  • Enter the email address of the person (or persons, if you enter a distribution list) that will be taking care of estimate requests from shoppers. The email address you enter should be the person (or persons) who can work with the shopper to complete their purchase. This is not the same as the persons who may contact the shopper if they select the "Contact Me" check box but the persons who can answer questions and provide assistance with the estimate.
