Serial Number Attributes

The following table outlines the attributes that can be set when serial numbers are enabled for a product.

Attribute

Definition

Generation Method

Indicates how the serial numbers will be distributed or generated to shoppers who purchase the product. This is not how the codes themselves are generated (they are always generated using a text file that you import).

Fixed Number – Used if you want all purchases of the product to use a single serial number. When you select this option, the "Fixed Number" input field appears in which you would enter the serial number all shoppers will use for this product.

Manual List – Used when you will enter or import a list of serial numbers to be used by the product. If you select this option you will have to set the other options that are explained in this table.
External – Used when you have created a custom integration to deliver serial numbers.

Distribution

Indicates how you want orders with multiple quantities of the same product to use serial numbers.

All Quantities Get the Same Number – Used if you want all quantities of the same product in an order to use the same serial number.

Each Quantity Gets a Unique Number – Used if you want all quantities of the same product in an order to use individual or unique serial numbers.

Serial Numbers per Unit

Number of serial numbers you want to use for each sale (individual unit) of the product.

Notification Email Address

Email address where the notification should be sent when the number of serial numbers for the product are running low.

Initial Warning Number

Number at which the notification should want to be notified that serial numbers are running low. When the available number of serial numbers for the product reaches this number, a notification is sent to the email address you provided.

Warning Notification Interval

Number of serial numbers that should be distributed is sent before another warning notification is sent (occurs after the initial warning). For example, if you enter "2" for the Warning Notification Interval, after an initial warning email is sent, another warning will not be sent until 2 more serial numbers for the product have been used/sent to a shopper. The warnings continue to be sent at these intervals until the serial numbers are depleted for a product.
