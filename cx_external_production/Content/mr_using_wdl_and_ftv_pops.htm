Using the Wait Don’t Leave and First Time Visitor Pop-Ups

If you want to create a "Wait Don’t Leave" or "First Time Visitor" offer, you must create two offers for the drop-in/pop-up to appear correctly on your store. The image below shows a sample of the two POPs you would create offers for if you wanted to create a "First Time Visitor" offer.

Both of the offers you create should use the "Always Triggered" trigger. The first offer should use the "activator" POP. This offer will only act as a activator for the real offer so you do not need to enter the offer display content, products, discount, and so on. You should immediately deploy the activator offer before you create the second offer. The second offer will use the regular POP and contain the products, pricing, and so on you want the offer to assume.
