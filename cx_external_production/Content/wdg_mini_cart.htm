Mini Cart Widget Version 2

Mini Cart Widget Overview

The Mini Cart widget shows a mini version of the shopping cart on the page where the JavaScript code is inserted. As a shopper adds and removes products to the shopping cart, the Mini Cart updates to reflect the contents of the cart.

You can use CSS to customize the look of the Mini Cart to match the style of your store. By default, the Mini Cart shows the product name, price, number of items in the cart, sub-total, and includes a Checkout button. Depending on how you set up the widget, you can include additional product details such as delivery method or platform.

How Mini Cart Works

When a shopper visits a page with Mini Cart, the loading progress bar displays until the Mini Cart is fully loaded on the page. If the cart is empty, a featured product with a description and image followed by a set of related or recommended products appears in the Mini Cart area. The featured and related products come from a marketing offer that uses the "SiteMerchandising_EmptyCart" point of promotion (POP). The first product set in the offer becomes the feature; the remaining products become the recommended products shown in the widget. The featured products information is displayed until a product is added to the cart.

When a product is added to the cart, a process bar indicates that the Mini Cart is updating. When the cart is done loading, the products are shown and a check out button is included so shoppers can complete their purchase from the Mini Cart.

Sample JavaScript

<script type="text/javascript" src="http://store.digitalriver.com/store/storeID/DisplayDRMiniCartJSPage/Version.2/dom.divID"></script>

To implement the mini cart widget:

  1. Make sure that a marketing offer has been created that uses the "SiteMerchandising_EmptyCart" POP and has at least one product associated with it. If the offer is set up correctly, selected featured and related products appear when the Mini Cart is empty.
  2. Copy the JavaScript below and paste it on the page where you want the widget to appear. You can specify exactly where the widget appears on the page using either of the following approaches:
    • Add the JavaScript in the exact position where you want the widget to appear. In this case, remove domID.divID/ from the JavaScript. 
    • Add the JavaScript anywhere to the page and replace divID with the name/ID of the div where you want the widget to appear. For example: <script type="text/javascript" src="http://store.digitalriver.com/store/storeID/domID.testDiv/DisplayDRMiniCartJSPage/Version.2">
  3. Replace the following elements in the JavaScript with the relevant information for your store:
    • Replace store.digitalriver.com with your Digital River store domain.
    • Replace storeID with your store ID.
  4. Add the optional includeCSS parameter to indicate whether you want to use the default CSS for the Mini Cart or your own CSS hosted on your store. To override the default Mini Cart CSS and use your own CSS, set includeCSS to no as shown in the following example: 

<script type="text/javascript" src="http://store.digitalriver.com/store/storeID/DisplayDRMiniCartJSPage/Version.2/includeCss.No/domID.testDiv"></script>
