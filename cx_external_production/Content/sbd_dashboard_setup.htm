Setting Up the Store Builder Dashboard

To get the Store Builder Dashboard to show your Google AnalyticsA third-party feature from Google that tracks activity on your store. Google Analytics can be enabled using the "add-ons" feature. data, you must first set up the Dashboard with your Google Analytics account.

To set up the Store Builder Dashboard:

  1. First, you will need some information about the Google Analytics™ account you want to set up to track.
    • If you already have a Google Analytics account, simply log on to your Google Analytics account and get the information listed in the next step.
    • If you do not have a Google Analytics account you must sign up for a new one using a Gmail email address. Once your account is set up, you can return here and complete these steps.
    • If you want to view site data already being tracked in another Google Analytics account, have the administrator for that account add your Gmail address to their account so that you can get your own Google Analytics username and password to enter in (Undefined variable: gc_cx_variables.PlatformName3). Make sure you can log on to Google and verify that you have been added to the account you want to track and that you can see that site's data in your Google account.
  2. Get the following information for the Google Analytics account. You'll need this information to set up the Dashboard.
    • Account ID (also called Property ID) – The account ID or property ID can be seen while viewing the Google Analytics account information. The Account ID should begin with "UA" followed by a string of numbers.

    • Profile ID – The profile ID can be seen when you view the profile settings for the site associated with the Google Analytics account.

  3. Enter your Google Analytics account information in Add-ons.
  4. Finally, enter the Google Analytics email address and password on the My Information page:
    1. To open the My Information page, click your user name in the upper-right corner of the screen.

    2. Click the Settings for [Your Store Name] tab.

    3. In the Email Address and Password fields, enter your Google Analytics login information. This information will allow the Command Console to log on to Google Analytics and retrieve data.

    4. Click Save to save the information you have entered. Once you save, the page should look similar to the one below.
