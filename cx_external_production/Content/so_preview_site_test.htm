Previewing Site Optimizer Tests

You can preview a Site Optimizer test and see how the different test levels appear (or will appear) when viewed on your store.

Types of Tests You Can and Cannot Preview

You can preview Site Optimizer tests for the following offers types:

  • Content
  • Product Discount
  • Order Discount
  • Product Shipping Discount
  • Order Shipping Discount
  • Cross-Sell
  • Up-Sell
  • Bundle

You cannot preview Site Optimizer tests for the following offer types:

  • A/B Links
  • Siteflow

Opening the Store Preview

  1. In the icon well, hover over the Preview Store option.

  2. From the Locale list, select the locale you want to view on your store. You should choose a locale that is also supported by the Site Optimizer test you want to test.
  3. For the Preview State option, select whether you want to view the design version or live version of your store. What you select here will not affect the status of the Site Optimizer test(s) you can preview– you will choose the status of the tests you want to preview in a later step.
  • Design – The design version of your store shows objects with "design" status, including any items that have a future start date (like an offer, or a product). The design status is given to any object that has been created and saved, but not yet been deployed to the store. If an object does not have a version with design status the version with deployed status will be seen.

Keep in mind that an object can have a version that is in design status and a version that is in deployed status. Design status does not mean the object has never been deployed, just that the specific version that is in design status has not been deployed.

Note: If you select to view styles, themes, content layouts, or offers with design status you will see those objects with either design status or deployed status. This includes any offers that have a start date set in the future (but not those that have expired).

If you select to view Site Optimizer tests with design status, you will see any test that has the following status: draft, pending start, and in progress. If you choose to view Site Optimizer tests with deployed status you will see any test with the in progress status.

  • Live – The live version of your store shows objects with "deployed" status. The deployed status is given to the last version of the object that was deployed to the store.

Note: If you cannot change a toggle from design to live (the toggle is set to Design and disabled) it means that there is not a deployed/live version of the object to preview.
  1. For the Test Order option, make sure Off is selected. To change what is selected for this option just click on toggle switch and the option will change.

  2. Click Launch Preview. A new tab or window will open showing your store with the options you selected.

Selecting the Preview Options

  1. Click the Preview Options tab that appears at the top of the page.


  2. Click the Site Tests tab.

  3. Make sure the Preview Tests option is set to On. When this option is set to on, all Site Optimizer tests that match the status you choose in the step below will be available for preview on the store.

Note: If the Preview Tests option is disabled it means that there are no Site Optimizer tests created for the site, or there are tests but they do not have one of the statuses that can be previewed.

  1. For the Available Tests for Preview option, select whether you want to view tests with design status or live status. Click the toggle switch to change what is selected for this option.
    • If you select to view tests with design status you can preview any Site Optimizer test that has the following statuses: Design, Pending Start, In Progress.
    • If you select to view tests with live status you can preview any Site Optimizer test that has the following status: In Progress
    • You cannot preview a test that has one of the following statuses: New, Pending Final Action, Completed, Completed (Promoted), Completed (Retired), Promoted, Retired, Complete (Expired)

Note: If you cannot change the selection of the Available Tests for Preview option it means that you do not have a Site Optimizer test that has the correct status needed to preview.

  1. From the lists next to the toggle, select the specific Site Optimizer test and then the level within that test that you want to view. If you do not see a test that you expect to see, make sure you have selected the right status for the types of tests you want to preview and that the test is not for the offer types that cannot be previewed.
  2. Click Update Preview to update the preview with the options you selected.
  3. Navigate to the page on your store where the Site Optimizer test can be viewed. You will see the test and level you selected in a previous step. When you are done previewing the test you can simply close your browser window.
