About Offer Triggers: Site Test

The Site Test offer trigger is only used with offers that you have created for Site Optimizer tests.

When you deploy an offer with the trigger of Site Test, the offer will not appear on your live store until it is part of a deployed Site Optimizer test. See About Site Optimizer for more information about using Site Optimizer to test marketing offers.

Offers That Can Use Site Test

The following offers can be used with the Site Test trigger:

  • Discount
  • Up-Sell
  • Bundle
  • Cross-Sell
  • Shopper-Defined Bundle
  • Banner
  • Featured Products
  • All of the shipping offers
