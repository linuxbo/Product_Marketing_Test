Import and Export Company Hierarchy

Your company hierarchy shows the structure of the sites in your company. It can be viewed in a spreadsheet file that you can make changes to, or in a text file that you can open and use for reference.

When you export your company hierarchy from the system you get a spreadsheet file with the company ID, sites in the company, and parent ID of the company. You can make changes to the data in the spreadsheet and import those changes into the system.

Note: The Company Hierarchy spreadsheet file has a file extension of .xls and can only be opened and edited using Microsoft®  Excel®. The Company Tree file has a file extension of .txt and can be opened with any standard text editor.

Exporting Company Hierarchy

You can export either your company hierarchy in a text file to be used as a reference, or in a spreadsheet file that you can edit and import back into the system.

The Company Hierarchy is exported in a spreadsheet file that you can use with Microsoft Excel. The Company Tree is exported in a text file that you can open and view with nearly any text editor.

To export company hierarchy:

Role Needed: Company ManagerA user role that provies access to several company-level features. Search this file for the "user role definitions" topic for more information.
  1. On the Administration menu, select Company, and click Import & Export Company Hierarchy. The  Import & Export Company Hierarchy page appears.
  2. In the Export Company Hierarchy area, select the type of file you want to export:
    • Select Blank Template if you want to enter the company hierarchy in a blank spreadsheet template that can import into the system when done.
    • Select Company Hierarchy (Spreadsheet) if you want to edit the existing hierarchy in a spreadsheet file and import those changes.
    • Select Company Hierarchy (Text) if you want to export a plain text file of the existing company hierarchy to use as a reference.
  3. Click Export. A dialog appears prompting you to open or save the file. If you open the file, make sure you save a copy if you make changes you want to import back into the system. If you save a copy of the file, make sure you save it in a location where you can find it later when needed.

Importing Company Hierarchy

If you exported your company hierarchy in a spreadsheet and mad changes, you can import those changes to the system. You can also import a new company hierarchy that you created using a blank template you exported.

To import company hierarchy:

Role Needed: Company ManagerA user role that provies access to several company-level features. Search this file for the "user role definitions" topic for more information.
  1. On the Administration menu, select Company, and click Import & Export Company Hierarchy. The  Import & Export Company Hierarchy page appears.
  2. In the Import Company Hierarchy area, click Browse and locate the file you want to import. This should be the spreadsheet you exported from the system previously and made changes or additions that you want to take effect for your company.
  3. Click Import. The file is processed and when the page refreshes any messages about the success of failure of the import appear on the page. If the import completed with no problems, your site is now updated with the information you imported. If there were errors you will need to fix the data in the spreadsheet before you can try the import again.
