Importing and Exporting Digital Rights

You can use the Import and Export tool to manage your serial numbers and unlock codes for a product. With the tool you can use a spreadsheet file to import serial numbers and unlock codes. You can also export unused serial numbers or unlock codes as needed.

Note: All imports and exports are processed in the background, asynchronously. This means you can start an import and export job, then work in other areas of (Undefined variable: gc_cx_variables.PlatformName3) as the job processes. When the job completes (or fails) you can view details about the job such as the warnings or errors that occurred, and view the file processed in the job.

To import digital rights:

Role Needed: Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information.
  1. On the Catalog menu, select Products, and click Import and Export Digital Rights. The Digital Rights Import and Export page appears.
  2. In the Import Serial Numbers/Unlock Codes area, for the Import Option, select what you want to import:
    • Serial Numbers - Local Purchases – Used to import serial numbers that are verified by (Undefined variable: gc_cx_variables.PlatformName3).
    • Serial Numbers - Activation Service – Used to import serial numbers that are verified by another service or system.
    • Unlock Codes – Used to import unlock codes.
  3. In the Product ID field, enter (or use the search feature to find) the product ID of the product for which the digital rights should be used. The Product ID may already be in this field if you came to this page from a product page.
  4. In the Job Name field, enter a name for the import or export job. The name you enter will be used to track the export in the Job Queue. To learn more about the Job Queue and why it's used, see The Import and Export Job Queue.
  5. Click Browse next to the File field and select the text file (.txt) with the digital rights you want to import. The size of the file you import is limited to 4 MB. Serial numbers and unlock codes can be of any length but must contain all Unicode Standard characters (except the paragraph return/new line character).
  6. For the Action option, select how you want the numbers in the file to be added to the list of existing numbers (if applicable).
  7. Click Import. The job is initiated and you can view the status in the Job Queue (click View Import Job Queue in the taskbar to go to the Job Queue and see the status of your job).

To export digital rights:

  1. On the Catalog menu, select Products, and click Import and Export Digital Rights. The Digital Rights Import and Export page appears.
  2. For the Export Option, select what you want to export:
    • Serial Numbers - Local Purchases – Used to export serial numbers that are verified by (Undefined variable: gc_cx_variables.PlatformName3).
    • Serial Numbers - Activation Service – Used to export serial numbers that are verified by another service or system.
    • Unlock Codes – Used to export unlock codes.
  3. In the Product ID field, enter the product ID of the product for which you want to export digital rights. The Product ID may already be in this field if you came to this page from a product page.
  4. In the Job Name field, enter a name for the import or export job. The name you enter will be used to track the export in the Job Queue. To learn more about the Job Queue and why it's used, see The Import and Export Job Queue.
  5. Click Export. The job is initiated and you can view the status in the Job Queue (click View Export Job Queue in the taskbar to go to the Job Queue and see the status of your job).
