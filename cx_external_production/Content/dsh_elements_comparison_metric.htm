Comparison Metric

When viewing Site Trends the Comparison Metric allows you to see how the performance of a particular Site Trend KPI (in the current timeframe) compares to the same time frame in the past.

For example, if Site Trends is showing daily data you can see how the KPI performed in the previous day, or in the same day last year. If Site Trends was showing weekly data you could see how the KPI performed in the previous week or in the same week last year.

The comparison option is available on both the summary and the detail view, however you can only change the time frame being reported in the detailed view.

Click here to see an example of the comparison data in the summary view. In this example, note how the Comparison Metric list is available so you can select either Same Day Last Year or Previous Day. If the view was showing weekly data, the options would change to allow you to select "Same Week Last Year" or "Previous Week."

The comparison periods (daily, weekly, monthly, and quarterly) are determined by the time frame you have selected in the detailed view. Until you change the time frame in the detailed view, the summary view will show daily data. Once you change the time frame in the detailed view however, the summary view will show data for that time frame until you change the time frame again.
