Viewing Orders: Transaction Detail Tab

The Order Summary page has five tabs: Order Summary, Products, Order Log, Transaction Detail, Notifications. These tabs show different information about an order and the customer who placed the order. To get to the Order Summary page, you must first search for and select an order to view (see Searching Orders for more information).

From the Order Summary page (and from any of the tabs on this page) you can perform tasks on the order such as resending the confirmation email, sending the shopper's password via email, and viewing the invoice given to the shopper for an order. You can also refund orders, reship products, complete orders, and so on.

  • To resend the digital rights or password, click Send DL/PW in the task bar.
  • To initiate a refund for the order, click Refund in the task bar.
  • To initiate a return for the order, click Return in the task bar.
  • To resend the confirmation email, click Resend Confirm Email in the task bar.
  • To send a shopper's password via email, click Send Password Email in the task bar.
  • To view the shopper's invoice for the order, click View Customer Invoice in the task bar.
  • To remove the customer's personal information from the order, click Remove Personal Info. For more about what happens when you click this button, see About Removing Personal Identifiable Information. Please note that you need the PII Removal Manager role to see or use this feature.
  • To complete an open order, click Complete Order in the task bar.

The Transaction Detail tab shows information about the transactions and payments for an order. The following sections outline the information that appears on this tab.

Transactional Data and Display Data

  • Total Authorized – The authorized transaction amount.
  • Total Canceled – The total canceled value for the order. This represents the portion of the order which was canceled.
  • Max Amount to Settle – This represents the maximum total amount of the order that can be settled.
  • Total Settled – Indicates the total amount which has been settled.
  • Total Refunded – Indicates the total amount which has been refunded to the shopper.
  • Total Cost – Indicates the total cost of the order.

Other Information

  • Transaction ID – Represents the unique identification number for the transaction.
  • Transaction Type – Represents the transaction type.
  • Card Type – Indicates the type of credit card used by the shopper, if applicable.
  • Card Number – Indicates the card number for the credit card used by the shopper, if applicable.
  • Expiration – Indicates the expiration date of the credit card used by the shopper, if applicable.
  • Request Amount – Indicates the amount requested for this transaction.
  • Response Amount – Indicates the amount that has actually been settled for this transaction.
  • Status – Indicates the status of the transaction.
  • Last Modified – Indicates the date when the transaction details were last modified.
