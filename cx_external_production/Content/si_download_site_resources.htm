Downloading Site Resources

You can download a Zip® file of all of the resource files used on your site. Site resources include items such as images and multimedia files. Downloading your site resources is useful if you want to download all of the image files currently on your site so that you can replace or delete some of the images in the Zip file and upload the new or changed Zip file.

To download resource files:

Role Needed: File Upload ManagerA user role that provides access to upload and download resrouce files in the Site Implementation and adminstration template features. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Site Implementation, and click Download Resource. The Download Site Implementation Resource Zip page appears.
  2. Make sure that the Site Name listed is the site you want to export the files from. If it is not, click the link that is the site name in the upper-right corner of the page to change the selected site.
  3. Click Download to download your site resources in a Zip® file. When prompted, save the file on your computer.
