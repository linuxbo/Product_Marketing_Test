Installing the ePen Report Macro

Microsoft® Excel® users may experience problems with their Detail ePen reports if the report contains double-byte characters. Digital River recently added UTF-8 encoding to ePen reports to support double-byte characters. Unfortunately, Microsoft® Excel® has does not recognize .csv files with UTF-8 encoding. As a result, data delimited by commas appears in a single column in the report.

The double-byte character problem only occurs with new reports. No data is compromised—this is only a formatting issue. To fix this problem, you must download and install a macro for Microsoft Excel that was created by Digital River.

To download and install the ePen macro:

  1. Click here to download the macro. When prompted, save the macro in a place you can easily find or remember.
  2. Open Microsoft® Excel® and install the macro.
    • Click here to show/hide instructions for Excel 2007.  
    • Click here to show/hide instructions for Excel 2003.
  1. In Excel, click the Office button in the upper-left corner of the screen and then click Excel Options. The Excel Options window appears.

  2. Click Add-Ins on the left.

  3. Make sure Excel Add-ins is selected in the Manage list at the bottom and click Go. The Add-ins window appears.

  4. Click Browse. Then, locate and select the macro you saved in the first step. When you select this file the Excel Add-in window refreshes to show the macro you just selected. Click OK to add the macro and close the window.

  5. After the macro is added, open an ePen report with the problem. Then, right-click in any cell of the report and click Text to Columns. The macro will run and reformat the report so that data is separated into columns. You must do this step in each new ePen report for the data to appear correctly.
