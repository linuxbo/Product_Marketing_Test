Creating a Post

Through SocialStream you can create social media posts, either basic or specific to an offer. Basic posts can—but do not have to—include existing offers.

Creating a post for a specific offer is done from with the offer itself, regardless of whether that is offer is a new one you are just creating or an existing offer you created in the past. Keep in mind that to create a new offer you must have the Merchandising Manager role (you can however, create a post for an existing offer without the Merchandising Manager role).

Creating a Basic Post

You can create a basic post in your social media account announcing new products or events on your store. If you want to create a post for a specific merchandising offer you have created, see Creating a Post for a Specific Offer below.

To create a basic post:

Role Needed: SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select SocialStream, and click Dashboard. The SocialStream Dashboard appears.
  2. On the taskbar, click Create Post. The Create SocialStream Post page appears.
  3. Select the social media accounts to post to.

Note: SocialStream can only add posts to Facebook pages (also called "fan pages," "pages for businesses" or "public profiles") not profiles. See About SocialStream for more about Facebook pages.

Creating a Post for a Specific Offer

Although you can include existing offers in basic posts, you'll often want to create a post specific to an offer from within that offer.  You can initiate a post specific to a new offer from the Offers list page immediately after deploying the offer. See Creating Marketing Offers to learn more about creating a new offer.

Note: You can also create a new offer from the SocialStream Dashboard or Post Log by clicking the Create Offer link in the taskbar, which will take you to the Create Offer page.

You can initiate a post specific to an existing offer from the Offers list page.

Note: If you have just created and deployed a new offer, on the Offers list page you will see a message indicating that your offer has been successfully created and deployed and can be posted to SocialStream. Click the Post this offer to SocialStream link in the message to create a post specific to the offer.

To create a post for an offer:

Role Needed: SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select Offers and Programs, and click Manage Offers. The Offers list page appears.
  2. Select an offer and click SocialStream. The Create Post page appears with a link to the selected offer at the top.
  3. Select the social media accounts to post to.

Note: SocialStream posts messages to Facebook pages, not personal profiles. Therefore, if your Facebook account does not include at least one page, you will not see a Facebook option. Follow the Facebook instructions to add one.

  1. Click Insert Tracking Link if you want to include a link in your post. The Create Tracking Link page appears.
  2. In the Post  field, enter you message. If you've created a tracking link, it will appear here.
  3. Click Preview to see how your post will look, or click Post to post your message to your social media accounts.
