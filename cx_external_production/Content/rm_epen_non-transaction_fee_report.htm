Non-Transaction Fee Report

The Non-Transaction Fee report contains the information listed below. Some of this data may not be calculated for the store because of the store set up and configuration.

Field

Definition

Charge Date

Date of the fee was charged.

Fee Type

Type of fee. For example, Setup Fee.

Description

Description of the fee.

Fee Amount

Amount of the fee.
