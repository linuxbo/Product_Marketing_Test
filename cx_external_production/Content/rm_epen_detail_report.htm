Detail Report

The Detail report contains the information listed below. Some of this data may not be calculated for the store because of the store set up and configuration.

Name

Definition

Customer ID

Unique ID automatically generated for the customer.

Customer Name

First and last name of the customer.

State

State where the order was shipped (to the customer).

Country

Country where the order was shipped (to the customer).

Order ID

Unique ID automatically generated for the order.

ProdNo

(Product Number) SKU associated with the product. SKU may not have been required, so this field may be blank.

Product Name

Internal Product Name set for the product.

Operating System

Supported operating system for the software (if applicable).

Gross Price

Total price for the line item (excludes tax, shipping, and discounts).

Discount

Total discount applied to the line item.

Net Price

Gross Price less Discount.

Net Cost

Amount due to client, excluding fees (Net Price less Digital River’s margin).

Fee

Total amount of Transaction Fees applied to the line item.

Remit Shipping

Amount of shipping paid to client by Digital River when they use a third-party fulfillment agency instead of DSS (a Digital River fulfillment partner).

Serial Number

Software serial number issued (if applicable).

Company Seller

Name of the selling company.

Site ID

Site/store ID of the selling store.

Fulfillment Date

Date when the order was fulfilled.

Dig Phy

(Digital/Physical) Indicates whether the product was a physical (phy) or digital (dig) product.

Locale

Locale for the transaction.

Region Code

Region where the sale took place.

Quantity

Number of products purchased. A sale is indicated by a one (1). A return is indicated by a negative one (-1). A refund is indicated by a zero (0) and has a negative Net Price and Net Cost.

Shipping Discount

The amount Digital River deducts from your payment for any free or reduced shipping promotions on your store. (Free or discounted shipping costs are absorbed by you, the client.)

Product VAT

VAT based on the product value. This is not used when calculating the amount due to client.

Shipping VAT

VAT based on the shipping costs. This is not used when calculating the amount due to client.

Unlock Code

Unlock code associated with the product in the transaction.

Program ID

Program ID (if any) associated with the transaction.

Offer ID

Offer ID (if any) associated with the transaction.

Product ID

Unique ID automatically generated for the product.

Variation Product ID

Unique ID automatically generated for the variation.

Product External Reference ID

External product reference ID (if any) associated with the product.

Product Type

Type of product.

Product Family

Product attribute family for the product in the transaction.
