Previewing Links

After you create a link, you can return to the Links tool in Store Builder and preview the destination for the link and see how it will behave when clicked by a shopper. To preview a link you simply select the link from the Links list page and click the Preview button.

  1. On the Storefront menu, select Links and Parameters, and click Manage Links.
  2. In the list of links, click the Name of the link you want to preview. The Edit Link page appears.
  3. Click Next until you reach the Summary page for the link. Then, on the Summary page, click Preview. A new window opens showing how the link you created will appear/function.
