About Files

The Files feature is used to import images and other resource files you want to use with custom pages and styles. These can include images for marketing landing pages, button images, CSS files, videos, and so on. Once you have imported your files, you can copy and paste the URL for the file into your custom page or reference the files in your style.

When you view the Files page you will see two tabs: Images and Resources. The Images tab is used to upload images for use on your store. The Resources tab is used to upload all other files for your store.

Depending on where you will use the files, you must select either the "Custom Pages" or "Themes & Styles" directory/folder before you upload. For example, if the file will be used with a style, you should select the "Themes & Styles" directory. Conversely, if the file will be used in a custom page, you would select the "Custom Page" directory. See Using Directories to Organize Custom Pages and Files for more about these directories.

Note: To use the Files features you must have the Page Designer or Content Manager role. For more information about roles and user profiles, see About User Roles.

Selecting the Right Location to Import

When you view the Files page you will see two tabs: Images and Resources. The Images tab is used to upload images for use on your store. The Resources tab is used to upload all other files for your store.

Depending on where you will use the files, you must select either the "Custom Pages" or "Themes & Styles" directory/folder before you upload. For example, if the file will be used with a style, you should select the "Themes & Styles" directory. Conversely, if the file will be used in a custom page, you would select the "Custom Page" directory. See Using Directories to Organize Custom Pages and Files for more about these directories.

Note: Take care when selecting the tab and folder location when you import files. You cannot move files once they are imported (but you can re-import a file into a different location), nor can you delete files once they are imported.

Supported File Types

There are several different types of image and resource files you can upload, outlined below. The type of files you can upload varies, depending on which tab on the Files page you are uploading from, and which folder you have selected on the tab.

The image and resource files you upload can be linked to and used by custom pages you create or upload to the system (see About Custom Pages for more information). To do this you would simply upload the file(s) you want to use, copy the path/ URL, and paste that URL in the custom page you want to reference the image or resource file.

Image Files

Images are uploaded on the Images tab on the Files page in Store Builder. Make sure you select the directory you want to upload the file in before you actually perform the upload. After you upload an image, you can preview the image and copy the image path/URL and paste that path in a custom page or other file.

The list below outlines the types of image files you can import:

  • .gif (Graphics Interchange Format)
  • .jpg or .jpeg (Joint Photographic Experts Group)
  • .png (Portable Network Graphics)
  • .bmp (Bitmap)
  • .ico (Icon)

Resource Files

Resources are uploaded on the Resources tab on the Files page in Store Builder. Make sure you select the directory you want to upload the file in before you actually perform the upload. After you upload a resource file, you can copy the file path/URL and paste that path in a custom page or other file.

The list below outlines the types of resource files you can import:

  • .css (Cascading Style Sheet)
  • .js (JavaScript)
  • .swf (Shockwave)
  • .pdf (Portable Document Format)
  • .flv (Flash Video)
