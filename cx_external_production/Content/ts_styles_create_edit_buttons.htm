Setting Button Images in Styles

You can change the default button images used in a style on the Buttons tab in Store Builder styles. The button images you can set include buttons used by shoppers to navigate your site and the shopping cart.

If you need to import images for buttons, you can import multiple images at the same by creating a Zip® file of the images and importing that file. The images you import in the Zip file will be available for you to select when you edit a button.

Note: Themes and styles are not available for stores that use the Headwater template. This means that if your store uses the Headwater template, you will not see the themes and styles options in the Store Builder menu as these Help topics indicate. To find out what template your store uses, contact your CMM.

Uploading Button Images from a Zip® File

To import button images using a Zip file:

Role Needed: Content ManagerA user role that provides access to various Store Builder features including themes and styles, liinks, files, compare pages, and Backup Media configurations. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, and click Manage Styles.
  2. Click on the Name of the style you want to edit.
  3. Click the Buttons tab.
  4. Click Browse next to the File field. Then, in the window that appears, select the Zip® file on your computer with the images you want to import.

  5. Click Upload. A message appears when the import is completed. The images you imported will be in the list of available files for you to select for a button when you set the button image (see the instructions below).

Changing Individual Button Images

When you view the Buttons tab, the current image set for a button appears next to the button name. You can change any or all of the images used by buttons on your site.

To change individual button images:

Role Needed: Content ManagerA user role that provides access to various Store Builder features including themes and styles, liinks, files, compare pages, and Backup Media configurations. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, and click Manage Styles.
  2. Click on the Name of the style you want to edit.
  3. Click the Buttons tab.
  4. Click on the button image you want to change.

Tip: Click Apply as you enter information or make changes to save and continue working on the page.

  1. When the Select an Image window appears, choose the image you want to use for the button.
To use the Select an Image window:

The Select an Image window is used to select an image, import a new image, and create folders in which to organize your images.

In this window you will see the images available for use on your site. The file name and size are shown for each and when you select a file in the list a preview of the image appears on the right. The width and height (in pixels) for the image are shown under the preview for easy reference.

Note: The exact size of the image you use for the category thumbnail, product thumbnail and product image may vary by the unique set up of your store, however most thumbnail images are 100px × 100px product images be 250px × 250px.

  • You can create folders to organize your files. Click New Folder and enter a folder name in the window that appears. You cannot use spaces and special characters in folder names, but hyphens (-) and underscores (_) are allowed.
  • To select a file to use, click the option button next to the image and click Add Selected.
  • To import a new file, click New File. The New File window appears. Click Browse to select the file you want to import. Select the Overwrite Existing File with Same Name if Present check box if you want the file to be saved over an existing image with the same name (useful when updating existing buttons with changes). Click Upload to import the file. When the import is complete you are returned to the Select an Image window.
  1. When you are done, click Save to return to the main Styles list.
