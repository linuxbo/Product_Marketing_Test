Retiring Content Layouts

You can retire a content layout when you no longer want to use the layout on your store. Content layouts that are retired can still be seen on the Content Layouts list page so you can create a copy of the layout if you want to create a new layout based on one that is retired.

Note: You cannot retire a content layout when it has been applied to a theme. See Using Themes to Apply Content Layouts to Pages for more about how content layouts can be used with themes.

To retire a content layout:

Role Needed: Page DesignerA user role that provides access to several Store Builder features incluidng links, files, and custom pages. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, and click Manage Content Layouts. The Content Layouts list page appears.
  2. Select the content layout you want to retire and click Retire. A message appears indicating that the content layout is retired.
