Setting Up a Reship Request

The Reship Request page appears when you click Reship Products on the Order Summary page. The list below outlines the fields and information you see on this page.

  • Billing Address – The billing address used by the shopper at the time the order was placed.
  • Email Address – The shopper's email address.
  • Shopper Login – The login ID the shopper used when placing the order. For orders that are not associated with an account, this is listed as anonymous.
  • Order Date – The date and time the order was placed.
  • Order Number – The identification number automatically assigned to the order.
  • Reason for Reship – Select a reason for the reshipment from the list.
  • Comments – Add applicable comments as necessary.
  • Shipment Information – If necessary, modify the shipping information for each shipment to be reshipped.
  • Shipping Method – The method the order was shipped by.
  • Reship All Products in Shipment? – Select this box to reship all of the items included in the original order.
  • Reship Products – Select individual items within the order to reship, and provide the quantity to reship.
