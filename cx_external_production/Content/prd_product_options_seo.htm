Search Engine Optimization (SEO)

The Search Engine Optimization settings are used to set up the text and keywords you want to be used when the product is searched for and viewed with a list of search results from a search engine such as Google™. The terms you enter for SEO may help raise the product in the search results listing, depending on the specific terms entered by the shopper for the search.

Search Engine Optimization Attributes

The following table outlines the attributes that can be set for products being set up for search engine optimization.

Attribute

Definition

Meta Description

Description that should appear under the "Custom Title" when the product is listed in search results.

Meta Keywords

Keywords you want to assign to the product. These are the keywords a shopper would have to enter in the search engine to see your product in the results. Keywords describe the product and should be phrases or words that appear in the product descriptions or names.

Title Tag

Title that should appear when the product is listed in the search results. The title appears as the primary link in the search results.
