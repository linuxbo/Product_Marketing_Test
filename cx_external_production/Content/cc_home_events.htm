Events

The Events pod on the Home page shows the events occurring today and tomorrow for the currently selected site. Today's events are listed individually and you can click on any one event to see more detail. Tomorrow's events are totaled with a link to jump to tomorrows date if more detail is needed.

Tip: Did you know that you can download individual events in iCalendar format and use those events in programs like Microsoft Office Outlook? See the information further in this topic for more information.

The Events pod shows just two days of events but you can click View Full Calendar to open the full version where you can see more details, change the view, and view events in other months. See example of the full calendar.

Note: The Events pod will not appear on the Home page if a site has not been selected from the site selector in the upper-right corner of the screen.

Using the Full Calendar View

When viewing the full calendar, use the controls on the top to change the timeframe you are viewing (week or month) and how events appear, or to scroll through months and jump to today's date.

When viewing events on the full calendar, you click an event link to open the item triggering that specific event (for example, the offer, link, SocialStream post, or product) or open a window showing a summary of the events.

Downloading Events

You can download individual events in iCalendar format and load those events into programs such as Microsoft® Office® Outlook®, Apple iCal, and Yahoo­® Calendar. This can be especially helpful if you set up reminders in these programs to notify you before these events occur.

To download an event, simply view the item with the event and look for the download link. You will find the download link near the attributes of the event, for example the start and end dates or the preorder settings.
