Custom Link Examples

To help you understand what you can do by building your own links, we’ve included several examples with an explanation of what each link is intended to do. We’ve highlighted, in bold, the elements in the URL that determine how the link operates. As you review these samples you may find it helpful to find the corresponding elements in the tables found in Creating Custom Links.

Warning! The examples in this section are for your reference only, and should never be used on your store. If you want to construct your own links, see Creating Custom Links.

Locale Links

You can use the "locale" element of the URL to have the link load the store using a supported locale. Keep in mind that locale indicates both language and currency.

http://store.digitalriver.com/store/demo/en_GB/home

Theme Links

You can use the "theme" parameter to indicate which theme you want to be used when the link is clicked and your store is opened. This is especially helpful when generating links for special promotions and shoppers. See Using Themes to Apply Styles to Pages for more about themes.

http://store.digitalriver.com/store/demo/DisplayHomePage/themeID.23200

"Buy" Links

The "buy" action allows you to create a link that will add a product to the Shopping Cart. With the "buy" action you can use additional parameters that define how many of the product are added to the cart, whether an offer is triggered by the action, whether the link is tracked by a program, what happens after the shopper clicks the link, and how the store or pricing appears to the shopper.

Add Specific Quantity of One Product

With the "buy" action, can use the "quantity" parameter to define how many of a specific product you want to be added to the Shopping Cart.

http://store.digitalriver.com/store/demo/buy/productID.45128400/quantity.2

Add Specific Quantity of Multiple Products

With the "buy" action, you can add multiple products to the cart and use the "quantity" parameter to define how many of each product should be added. Make sure you always put the "quantity" parameter right after the "productID" parameter or the link will not work correctly.

http://store.digitalriver.com/store/demo/buy/productID.45128400/quantity.1/productID.46143500/quantity.2

Add Product in Marketing Offer

With the "buy" action you can use the "offerID" parameter to have the link trigger an offer created for the product being added to the Shopping Cart.

http://store.digitalriver.com/store/demo/buy/productID.8688900/offerID.107138209

Add Product and Use Program ID to Track

With the "buy" action, you can use the "programID" parameter to indicate you want the traffic from the link tracked by a program you have already created for the store.

http://store.digitalriver.com/store/demo/buy/productID.8641700/pgm.215857
