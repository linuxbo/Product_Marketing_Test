Digital River Suite: Adding an Application

You can add your credentials to one of the Digital River Suite applications. This allows you to access these systems with just one click.

Opening the Digital River Suite Ribbon

  1. In the upper-center of the screen, click the Digital River Suite link. The Digital River Suite ribbon opens.

Opening the Add Application Window

  1. In the Digital River Suite ribbon, click Add Application for the application that you want to add. The Add Application window appears.

Entering Your Credentials

  1. Make sure the application selected in the Application list is the system for which you want to enter your logon credentials.

  1. In the User Name field, enter your user name for the system for which you are adding.
  2. In the Password field, enter your password for the system for which you are adding.
  3. Click Save. The credentials are saved and you can now access the system with one click the Digital River Suite ribbon.
