Editing Advanced CSS in Styles

You can change the definitions in the Cascading Style Sheet (CSS) used with a style on the CSS tab in styles. CSS is a stylesheet language that is used to set the look and feel for documents written in many languages, including HTML and XHTML.

A CSS defines elements on your site like fonts, colors, page layout, tables, and other elements that affect how the pages of your site appear to shoppers. We do not recommend that you change the CSS for a style unless you are familiar with CSS. Visit the CSS pages on the World Wide Web Consortium (W3C) Web site for more about CSS.

Note: Themes and styles are not available for stores that use the Headwater template. This means that if your store uses the Headwater template, you will not see the themes and styles options in the Store Builder menu as these Help topics indicate. To find out what template your store uses, contact your CMM.

To change the default CSS:

Role Needed: Content ManagerA user role that provides access to various Store Builder features including themes and styles, liinks, files, compare pages, and Backup Media configurations. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, click Styles.
  2. Click on the Name of the style you want to edit.
  3. Click the Advanced CSS tab.
  4. If it is not already selected, select the Use Custom CSS check box to allow you to edit the default CSS. Clear this check box if you want to use the default CSS set up for the site.
    • If you select the Use Custom CSS check box, a message appears informing you that your changes to the CSS may impact how the style appears. Click OK to continue to define your own CSS, or click Cancel to use the default CSS as is.
    • If you clear the Use Custom CSS check box, a message appears informing you that you will lose any changes you made to the CSS and that it will return to the pre-defined default. Click OK if you wish to continue and do not care about losing any changes you've made, or click Cancel to continue using a custom CSS that you define.
  5. In the CSS field, make your changes to the CSS definitions as needed.

Tip: Click Apply as you enter information or make changes to save and continue working on the page.

  1. When you are done, click Save to return to the main Styles list.
