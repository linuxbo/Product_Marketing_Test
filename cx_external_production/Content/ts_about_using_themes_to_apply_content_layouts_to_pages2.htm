Using Themes to Apply Content Layouts to Pages

A theme is a collection of styles that you apply to some of all of the pages on your store. Themes not only control what styles are used by pages and locales, but whether the "Continue Shopping" button appears in the Shopping Cart, where it goes when clicked, and the technical support email address shoppers see on your store.

Content layouts are used to customize the content and design of pages for your store, including checkout and product or category detail pages. After you create a content layout you can then assign that content layout to various pages on your store using a theme. The Apply Styles tab is where you define the theme by choosing which locales and pages should use styles (or content layouts, see Using Themes to Apply Content Layouts to Pages) you have created for your store.

Note: Themes and styles are not available for stores that use the Headwater template. This means that if your store uses the Headwater template, you will not see the themes and styles options in the Store Builder menu as these Help topics indicate. To find out what template your store uses, contact your CMM.

To apply a content layout to pages using a theme:

Role Needed: Page DesignerA user role that provides access to several Store Builder features incluidng links, files, and custom pages. Search this file for the "user role definitions" topic for more information.
  1. First, create the content layout you want to apply to a page on your store (see Creating and Editing Content Layouts for more about how to create and edit content layouts. There are several different pages for which you can create a content layout (see About Content Layouts for more about the pages you can customize).
  2. Once you have created the content layout you want to apply to a page on your store, you are ready to create or open the theme you will use to apply content layouts to pages on your store. For help creating a theme, see Creating Themes.
  3. While viewing the theme, click the Apply Styles tab.
  4. Find the store page you want to apply a content layout to and then select the content layout you have created for that page. Keep in mind that you must have a content layout created for the specific page type you want to apply a new layout to or the list where you select a content layout will not appear.
    • If "Inherited" appears in the Associated Layout column it means a content layout hasn't been created for page type or you cannot apply a content layout to that page.
    • If a list appears in the Associated Layout column it means you can create a content layout for that page and apply it within a theme. If the content layout selected in that list begins with the word "default" it means that the page is currently using the default layout from the template.

    In the example below, you can see that a new content layout called "My New Content Layout" can be selected as the layout for the "Three Page Confirm Order Page."

  5. Repeat the step above for any additional pages you wan to customize with a content layout. When you are done selecting content layouts for your pages, click Save.
