Basic Steps to Implement a Site Optimizer Test

Creating and running a successful Site Optimizer test involves three basic steps, as summarized below. See the topics indicated to learn more about performing each step.

To implement a Site Optimizer test:

  1. Create the offers or gather the content to be tested. When you create the marketing offers for your test, you must select "Site Test" as the trigger or the offer will not be available in Site Optimizer. The rest of the offer set up will vary, based on the type of test you want to perform and how you want the offers in the test to behave. For more information about creating and using offers, see the information that begins in About Marketing Offers. To learn which types of offers are used for the various test types, see Offer Types and Test Types.
  2. Create the Site Optimizer test. Tests are created using the New Site Optimizer Test wizard, which walks you through each step of the creation process. During the creation process you will have to select the offers or content you want to test, as well as define some other things about the test. Don't forget to deploy the test or it will not run on your store.
  3. Monitor the test and promote the champion. Tests can be viewed while they are in progress and stopped at any time by promoting a winning test level or ending the test. You do not have to promote a winner to end a Site Optimizer test.
