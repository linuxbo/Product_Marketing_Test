Business Navigator: Merchandising Categorization

Merchandising Categorization reports show sales, attach rates, and close ratios for bundle, cross-sell, and up-sell offers.

Report Name Description
Cross Sell Conversion Detailed report showing how cross-sell offers turn into orders.
Cross Sell Parent Child Attach Detailed report showing parent product attach rates, by products.
Up Sell Parent Child Attach Report Detailed report showing how up-sell offers turn into orders.
Up Sell Attach Rate Detailed report showing how up-sell offers turn into orders.
