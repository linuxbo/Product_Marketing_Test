Viewing Invoices

The Order Summary page contains five tabs (Order Summary, Products, Order Log, Transaction Detail, Notifications) that show different information about an order and the customer who placed the order.

You can reach the Order Summary page by first searching for orders, then in the results that appear clicking the Order Number for the order you want to view.

From the Order Summary page (and from any of the tabs on this page) you can perform tasks on the order such as resending the confirmation email, sending the shopper's password via email, and viewing the invoice given to the shopper for an order. You can also refund orders, reship products, complete orders, and so on.

  • To view the shopper's invoice for the order, click View Customer Invoice in the taskbar.
