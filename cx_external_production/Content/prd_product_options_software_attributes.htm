Software

The Software setting is used to define computer software products. This includes information about the system requirements, license agreement, video card requirements, supported platform, and so on.

EULA

A EULA or End User License Agreement is a legal agreement between the you (or the manufacturer) and the shopper who purchases your software product. When you create a software product you can enter the EULA text and have that text appear somewhere on your store. How, when, and where the EULA appears to shoppers varies by how your store is set up to show the EULA text. For example, it may appear on the Product Details page or on an interstitial page during checkout. To learn more about where the EULA appears to shoppers, contact your Store Operations team.

Restricting the Platform List

The Platform list is used in the Software setting to indicate which platform the software product is compatible with. You can restrict which items appear in this list by customizing the Product Template. See Configure Product Setup: Global Attributes for more about how to customize the Platform list.

Software Requirements

The various requirements you can enter for a software product are usually set to appear on the Product Details page for the product. You should only enter requirements for those specific requirements that apply to your software product. To learn how and where these requirements appear on your store, contact your Store Operations team.

Software Attributes

The following table outlines the attributes that can be set for software products.

Attribute

Definition

EULA (End-User License Agreement)

Text of the license agreement shoppers must accept before they can download the software.

Minimum Hard Disk Space Required

Lists the minimum space required on a hard drive to install the software.

Minimum Memory Required

Lists the minimum memory (RAM) required to use the software.

Minimum Processor Speed Required

Lists the minimum processor speed required to install or use the software.

PAD File

The Portable Application Description (PAD) file that contains the product descriptions and specifications. PAD files are XML formatted and used to provide product information in a standardized format for online use.

Platform

Indicates the platform the software supports.

Product Detail Image Notes

Text you want to appear below the product image seen on the Product Details page. Most stores are not set up to show this text. Contact your Store Operations team to learn if your store uses this attribute or how you can enable this attribute for your store.

Search Engine Optimization

Similar to the "Search Engine Optimization" setting, this attribute is used to set up text you want seen when the product is viewed in the results page of a search engine such as Google™.

System Requirements

Indicates the system requirements to run the software. This text appears on the Product Detail page in your store.

Thumbnail Image Notes

Text you want to appear below the thumbnail image when it is seen on your store. Most stores are not set up to show this text. Contact your Store Operations team to learn if your store uses this attribute or how you can enable this attribute for your store.

Video Card Requirements

Lists the video card requirements to use the software.
