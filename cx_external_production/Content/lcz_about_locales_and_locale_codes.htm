Locales and Locale Codes

In general, a locale can be thought of as a geographical area such as a nation, region, or territory. In terms of your store, a locale is any country where your shoppers are located and sales will occur, and any country that you want to specifically support on your store by showing pricing and store content in that country’s currency and language.

A locale code is a four-character code that represents the country and language for a specific place. The first two characters in a locale code are the language code. The language code is always lowercase and usually an abbreviation of the language the code represents. The second two characters in a locale code are the country code. The country code is always uppercase and usually an abbreviation of the country the code represent. For example, the locale code "en_US" represents the United States because "en" is the language code for English and "US" is the country code for the United States.

Tip: Locale codes consist of the official language code and country code as assigned by the International Organization for Standardization (ISO). Visit the ISO web site at www.iso.org to learn more about the ISO.

Locales with Multiple Countries or Languages

Locales are not limited to a single country or language. A country can have more than one locale (some countries may even share locales) and a language can be used by several different locales. It is the unique combination of language and country that help define the locale, and therefore the currency of the locale.

For example, in Canada there are two official languages: English and French. Thus, Canada has two major locale codes: "en_CA" and "fr_CA." Shoppers from Canada may be able to shop your store using the "en_US" locale because this locale uses English, but to fully support shoppers in Canada you would localize your store for both locales in Canada.

Locale Codes in URLs

When you are viewing your store, the URL for the page reveals what locales you are seeing. Within the URL you may see something like "Locale=en_US" or "Locale=es_ES." This is known as the locale code or locale parameter and it is this part of the URL that dictates what content is seen on the page. For example, changing the locale parameter in the URL from "en_US" to "de_DE" would change the language from English to German and the currency from dollar to euro (if your product and store content is localized).
