Creating Custom Pages

The custom pages feature allows you to create static body content for storefront elements like landing pages for marketing offers, or pages containing static content about your company. To learn how to upload a custom page, see Uploading Custom Pages.

To create a custom page:

Role Needed: Page DesignerA user role that provides access to several Store Builder features incluidng links, files, and custom pages. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, and click Manage Custom Pages.
  2. Click Create Custom Page. The first step of the Create Custom Page wizard appears.
  3. Look at the path next to the Filename label and make sure this is where you want your custom page created. To change the path, click Select Directory and then create or select the directory (folder) where you want the custom page created.
  4. In the Page Name field, enter a name for the custom page. Custom page names cannot contain spaces.
  5. In the Description field, enter a description for the custom page. Descriptions are not required or seen by shoppers on your site.
  6. In the Supported Locales area, select the locales you want the custom page to support. Make sure you designate a locale as the default. The default locale is used when a shopper from a locale not selected tries to access the page.  
  7. Click Next.
  8. Along the left you will see tabs for each locale you have selected. In the HTML field, enter or edit the HTML code for the page for that locale. You do not need to enter the <body> tags as these are automatically added for you.

Tip: For information about the HTML code you can use in HTML Content pages, see Using HTML in the Command Console.

To add a token:

  1. Click the Available Tokens link. The Tokens window appears.
  2. Click in the field of the token you want to use. Note how the contents of the entire token field is highlighted.

  3. Press CTRL+C on the keyboard to copy the token to the clipboard (you can only copy one token at a time) and click Done to close the window.
  4. Place the cursor where you want to insert the token you just copied. Then, press CTRL+V to paste the copied token code from the clipboard.
  5. Repeat the steps above to add additional tokens.
  1. Below the HTML field you will see the Search Engine Optimization fields. These fields are used to enter SEO information for the custom page:
    • In the Title Tag field, enter the title that should appear when the custom page is listed in the search results. The title appears as the primary link in the search results.
    • In the Meta Description field, enter the description that should appear under the Title Tag when the custom page is listed in search results.
    • In the Meta Keywords field, enter the keywords that you want to assign to the custom page. These are the keywords a shopper would have to enter in the search engine to see this custom page in the results. Keywords describe the contents of the custom page and should be phrases or words that appear on the custom page.

    2. Note: If your store has disinherited the “TopHeader” or “TopHeaderMetaAndTitle" elements of your template, SEO will not work on your store. If you cannot edit the SEO settings it means your store has disinherited these elements or you do not have the correct roles to change these settings.


  1. If you are editing a custom page, you can create a link to the page by clicking Create Link in the taskbar. When you click Create Link you are taken to the second step of the Create Link wizard and certain attributes are already selected for the link (for example, the page you were editing and wanted to create a link to is selected as the page for the link). See Creating Links for more steps about creating the link.
  2. When you are done entering the HTML for the custom page, click Next. The Review step of the wizard appears. Review the custom page you have created and when ready, click Finish to create the custom page.
