Creating Styles

A style is a collection of design elements such as colors for links, button images, font style and size, and so on. Styles are used to create themes, and themes can be applied to pages on your site.

Note: Themes and styles are not available for stores that use the Headwater template. This means that if your store uses the Headwater template, you will not see the themes and styles options in the Store Builder menu as these Help topics indicate. To find out what template your store uses, contact your CMM.

When you edit a style that is already deployed, the changes are saved with a new version number (the last five versions are saved) and in the "design" status. You must deploy any changes before they can be seen on your site even if the style you change is part of an active theme on your site.

When you edit a that has previously been deployed, the Version list appears. From this list, select the version of the style you want to edit and then click the Edit button to unlock the style and begin making changes.

To create a style:

Role Needed: Content ManagerA user role that provides access to various Store Builder features including themes and styles, liinks, files, compare pages, and Backup Media configurations. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, and click Create Style.
  2. In the Name field, enter (or edit) the name for the style. Names are required but they are only seen be you, not by shoppers on your site.
  3. In the Description field, enter (or edit) the description for the style. Descriptions are not required or seen by the shopper.
  4. Set the options for the style on the tabs on the page. Click a tab to set or edit the options for that attribute of the style. See the topics below for more help about working on a specific tab:

Tip: Click Apply as you enter information or make changes to save and continue working on the page.

    • Setting Colors and Fonts in Styles – Learn how to change the font and size for the text on your site. Also change the color of headings, hyperlinks, tables, and the background for various parts of a page.
    • Setting Button Images in Styles – Learn how to change the image used for buttons that appear on your site and in the shopping cart. Also learn how to upload a Zip file of multiple images at once.
    • Editing HTML in Styles – Learn how to edit the HTML used by pages on your site.
    • Editing Advanced CSS in Styles – Learn how to edit the CSS used by pages on your site.
  1. When you are done, click Save to return to the main Styles list, or click Apply to save the changes you have made and continue creating or editing the style.
