Creating and Editing Rules

Rules allow you to define which marketStream products from the Common Catalog you want to add to your catalog and sell on your site. Products are automatically added to your catalog based on the criteria that defines the rule.

For example, you can create a rule that includes products of a certain cost range or exclude products that have been assigned to a specific category. You can create as many rules as needed to specifically target the products you want to include or exclude.

Rules are executed on the order in which they appear in the list and you can change the order of rules as needed. See Reordering Rules for more information.

To create a rule:

Role Needed: Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information.
  1. On the Catalog menu, select marketStream Channel, and click Manage Rules. The Customize Rules page appears.
  2. Click Create Rule in the taskbar. The Edit Rule window appears.
  3. From the Criteria list, select the criteria you want to base the rule on:


    • Margin – Used to create a rule based on the revenue margin for a product. This is the amount retained by Digital River.
    • Cost – Used to create a rule based on the cost to the product seller. When you create a rule by Cost, you must also select the currency you want to use with the rule. This is the amount the product seller will be charged by default. This amount can be changed by editing the marketStream product pricing (see Editing marketStream Pricing). The difference of cost and margin is the amount retained by the product seller.
    • Category – Used to create a rule based on the product category. When you create a rule by Category, you select the category you want to use to define the rule from a drop-down list that appears in place of the Value field.
    • Publisher – Used to create a rule based on who is the publisher of a product. In marketStream programs, publishers can also be thought of as the Product Owners of marketStream products.
  4. From the Operator list, select the operator for the rule. The operators that you can select vary by the Criteria you selected in the previous step.
  5. For Value, enter or select a value for the rule you are creating (depending on the Criteria for the rule you may be able to select a value from a list or enter a value in a text field):
    • Use numbers and/or letters to set the rule Value (unless creating a rule to filter by Category, in which case you select the category from the drop-down list).
    • Depending on the type of rule you are creating you may need to select a currency from a list below the Value field.
  6. To exclude products that match the rule value, select the Exclude check box. Products that match the rule you are creating will not be added to your catalog and cannot be sold on your site if you select this option.
  7. Click Save. The rule is saved and the Customize Rules page appears. If you created a new rule it appears in the list.
