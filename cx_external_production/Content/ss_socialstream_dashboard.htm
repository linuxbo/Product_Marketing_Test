Using the Dashboard

The SocialStream Dashboard displays key sales data specific to sales originating from SocialStream posts.

Note: Before you can view the SocialStream Dashboard and begin using SocialStream, a representative for your site must accept the SocialStream Service Agreement and a minimum of one social media account must be set up and active.

Opening the Dashboard

Role Needed: Merchandising ManagerA user role that provides access to various merchandising features, including offers and tracking programs. Search this file for the "user role definitions" topic for more information. or SocialStream ManagerA user role that provides access to SocialStream. Search this file for the "user role definitions" topic for more information.
  • On the Marketing menu, select SocialStream, and click Dashboard. The SocialStream Dashboard appears. If you have the SocialStream Manager role, you will have the ability to manage your posts and accounts via the Dashboard. If you only have the Merchandising Manager role you will only be able to see how current social marketing efforts are performing.

What You See on the Dashboard

The following outlines what you will see when you are viewing the Dashboard.

  • The number of Twitter Followers appears for each Twitter account.
  • The number of Facebook Fans/Likes appears for each Facebook account.
  • SocialStream account activity data for the past seven days appears in the SocialStream Performance area.
  • The default view when accessing Dashboard is for All Accounts. If only one social media account is set up, than All Accounts is the only option available.  The available options are:
  • All Accounts (default view when accessing screen) — this will display all social media accounts set up.
  • All Twitter — Shows all Twitter social media accounts .
  • Each Twitter account — Shows reporting for specific Twitter social media accounts.
  • Each Facebook account — Shows reporting for specific Facebook social media accounts.
  • The following account activity data appears:
    • Total Sales – Gross sales for orders shipped that are directly a result of users clicking on a SocialStream link.
    • Average Order Value – Sales divided by number of orders that are directly a result of users clicking on a SocialStream link.
    • Revenue Per Visitor – Sales divided by number of unique visitors per post.
    • Close Ratio – Number of completed sales per unique visitors per post.
