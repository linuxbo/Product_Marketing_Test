Creating Tracking Programs

Tracking programs are used to track the performance of offers and marketing campaigns for your store. With a tracking program, you can see how many orders resulted from shoppers responding to a offer, banner ad, email promotion, and so on.

After you create a tracking program you can use the tracking program ID in links you create for offers you have created for your store. See Creating Links for instructions on how to create a link.

Note: We do not recommend editing a tracking program that is currently in use by links for your store as that will affect the reporting of usage for the link.

Role Needed: Merchandising ManagerA user role that provides access to various merchandising features, including offers and tracking programs. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select Offers and Programs, and click Create Tracking Program. The Create Tracking Program page appears.
  2. In the Name field, enter or edit the name of the tracking program (for example "Holiday Promotion 1"). Names are used internally to keep track of tracking programs and are not seen by shoppers on your site.
  3. In the Description field, enter or edit the description of the tracking program. Descriptions for tracking programs are required but are not seen by shoppers on your site.
  4. From the Type list, select the type of tracking program you are creating. This option indicates how the link will be distributed to shoppers. This option is only used for tracking purposes and does not affect how the tracking program is used or operates.
    • Email – The tracking program URL will be sent to shoppers via email.
    • Banner – The tracking program URL will be attached to a banner.
    • Navigation Analysis – The tracking program URL will be in tags within your site.
    • Offline – The tracking program URL will be displayed in printed media or a Google word search.
  5. Click Save.
