Searching Shoppers

You can search for shoppers using the shopper ID, first and last name, or email address.

Note: When conducting a shopper search using a shopper's name, you must type both a first name and a last name.

To search for shoppers:

  1. On the Customer Service menu, select Search, and click Search Shoppers. The Search Shoppers page appears.
  2. Enter criteria in the Login ID, First Name and Last Name, or E-mail Address fields.The criteria you enter in these fields must match a shopper exactly, except for the First Name field. In this field, you can enter just the first few letters of a shopper's first name to find matches.
  3. Click Search. The page refreshes and the search results appear on the Shopper Search Results list page. Click on the Login ID link for a shopper to view or edit the shopper details.
