Using the Product Editor:

Adding and Removing Variations

Using the Product Editor you can add or remove variations to an existing product. If you add variations to an individual product it will automatically convert the product to a multiple product with variations.

Tip: To see the name and ID of a variation, hover the cursor over the variation in the first pane. A tooltip will appear showing the ID and name for the variation.

To add or remove variations in the Product Editor:

Role Needed: Catalog ManagerA user role that allows the user access to the catalog and product features. Search this file for the "user role definitions" topic for more information.
  1. Click the Add/Remove Variations link under the first pane on the Details tab of the editor.

  2. To create variations:

    In the Create Variation area, select how you want to vary the base product to create variations and click Create or Create & Edit.
    1. If you click Create, the variation will simply be created and you will have to enter details for the variation at a later time.
    2. If you click Create & Edit, the variation will be created and you will be taken back to main Product Editor window.
  3. To delete variations:

    In the Existing Variations area, select the variations you want to delete and click Delete.
  4. To set the ordering for variations:

    Click Ordering. In the window that appears, click the arrow icons to move the variations up or down in the list. The order in which they appear on this screen is the order they will appear when listed on your site.
  5. To change how the base product is varied:

    First, delete all the existing variations for the product. Then, click Edit Variation Options. In the window that appears, select the attributes on which you want the base product to vary.
