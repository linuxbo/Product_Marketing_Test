Copying Backup Media Configurations

Copying an existing configuration is a quick way to create a new configuration for your store.

To copy a Backup Media configuration:

Role Needed: Backup Media ManagerA user role that provides access to create Backup Media configurations. Search this file for the "user role definitions" topic for more information.
  1. On the Marketing menu, select Backup Media Services, and click Manage Configurations. The Backup Media Configurations page appears.
  2. Select the configuration you want to deploy and click Copy. The screen refreshes and the copy you just made appears in the list, with the words "Copy of" before the configuration name. You can now edit the copy as needed, and deploy it as your normally would a configuration you created from scratch.
