Deploying Custom Pages

When you are done creating, editing, or importing a custom page you should deploy it so it can be seen or used on your store. When you deploy a custom page it becomes available for use on your store. If you do not deploy a custom page you will not be able to see the page on your store or even link to the page

To deploy a custom page:

Role Needed: Page DesignerA user role that provides access to several Store Builder features incluidng links, files, and custom pages. Search this file for the "user role definitions" topic for more information.
  1. On the Storefront menu, select Store Builder, and click Manage Custom Pages.
  2. Select the custom page(s) you want to deploy.
  3. Click Deploy. The custom page is deployed.
