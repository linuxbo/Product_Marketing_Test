About Links

The Links feature is used to create hyperlinks that add a specific product to the shopping cart, take the shopper to a specific product page, open your store’s home page, or open a custom page or external Web page.

Why Create Links?

Links are most often created for use with marketing campaigns, offer emails, and custom pages you have created or imported to your site. Links can trigger an offer you have already created or they can use a program ID to track how many shoppers click the link. After you create the link you can send it to shoppers via email, attach it to an image or button on your store, and so on.

Note: To use the Links feature you must have the Content Manager, Page Designer, or Merchandising Manager user roles assigned to you in your user profile. For more information about roles and user profiles, see About User Roles.

Creating Links for Products

When you create or use a link that adds a product to the shopping cart or takes the shopper to a specific product page you must make sure that the link is/was created for the version of the product that is currently deployed on your store. If the link was created for a previous version it may not behave as you expect and may cause problems with shoppers placing orders for the product or trying to use the link as you intend. This means that if you change (and the deploy those changes) a product for which you have created links you recreate or update the link.

To learn more about product versions (and how the version of a product can change) see About Product Versioning.

Links for Specific Themes, Locales, and Currencies

When you create a link, you can specify if you want the link to show your store using a specific theme you have already created. You can also indicate if you want the link to load your store using a specific locale or use a specific currency to display any pricing information. This can be especially useful when creating marketing campaigns for shoppers in specific locales because the store will be tailored for the language and currency of the shoppers you’re targeting.

Note: If you specify a locale or currency for the link, the locale and currency must be supported by your store to work as you intend. Links created for a specific locale or currency that are not supported on your store will load using your store’s default locale and currency.
